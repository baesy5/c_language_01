    여러 선택지 깔끔하게 처리하기 — switch문과 게임 메뉴¶

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • switch문의 구조와 동작 원리를 이해하고 직접 작성할 수 있다
    • switch문과 if-else if문의 차이를 비교하고 적절한 상황에 선택할 수 있다
    • break의 역할을 이해하고 빠뜨렸을 때의 결과(fall-through)를 예측할 수 있다
    • 게임의 메뉴 선택 화면을 switch문으로 구현할 수 있다

    예상 학습 시간: 약 1시간 30분 ⏱️

    📚 미리 알아야 할 것¶

    이전 장들에서 배운 내용 중 이번에 꼭 필요한 것들이에요:

    • if-else if-else문: 조건에 따라 다른 코드를 실행하는 방법
    • scanf()로 사용자 입력 받기: 키보드에서 숫자나 문자를 입력받는 방법
    • while 반복문: 조건이 참인 동안 코드를 반복하는 방법
    // 이전에 배운 if-else if 패턴 (기억나시죠?)
if (choice == 1) {
    printf("새 게임\n");
} else if (choice == 2) {
    printf("이어하기\n");
} else if (choice == 3) {
    printf("설정\n");
} else {
    printf("잘못된 입력\n");
}

    이번 장에서는 이 패턴을 더 깔끔하게 쓰는 방법을 배웁니다! 😊

    💡 왜 이걸 배우나요?¶

    게임을 떠올려 보세요. 게임을 시작하면 가장 먼저 보이는 화면이 뭔가요?

    바로 메뉴 화면이죠!

    ╔═══════════════════╗
║   🎮 나의 RPG 게임   ║
║                   ║
║   1. 새 게임       ║
║   2. 이어하기      ║
║   3. 설정         ║
║   4. 종료         ║
║                   ║
║   선택하세요 >>    ║
╚═══════════════════╝

    사용자가 1, 2, 3, 4 중 하나를 누르면 각각 다른 동작을 해야 합니다. if-else if로도 만들 수 있지만, 선택지가 5개, 10개로 늘어나면 코드가 점점 복잡해져요.

    switch문은 이런 상황에 딱 맞는 도구입니다. 코드가 깔끔해지고, 어떤 선택지가 있는지 한눈에 보여요!

    📚 핵심 개념¶

    개념 1: switch문의 기본 구조 🔢¶

    비유로 시작¶

    switch문은 엘리베이터와 같습니다.

    엘리베이터에서 버튼 3을 누르면 3층으로 직행하죠? 1층, 2층을 거치면서 "여기가 맞나?" 확인하지 않습니다. 누른 번호에 해당하는 층으로 바로 이동합니다.

    switch문도 마찬가지예요. 주어진 값을 보고, 그 값에 해당하는 case로 바로 점프합니다.

    flowchart TD A["switch(choice)"] --> B{"choice 값은?"} B -->|"1"| C["case 1: 새 게임"] B -->|"2"| D["case 2: 이어하기"] B -->|"3"| E["case 3: 설정"] B -->|"그 외"| F["default: 잘못된 입력"] C --> G["switch 끝"] D --> G E --> G F --> G

    반면 if-else if는 계단을 하나씩 올라가는 것과 같아요. 1층인가? 아니면 2층인가? 아니면 3층인가? 하나씩 확인합니다.

    정확한 정의¶

    switch문은 하나의 변수(또는 식)의 값을 여러 case와 비교하여, 일치하는 case의 코드를 실행하는 다중 분기문입니다.

    📖 다중 분기문: 여러 갈래 중 하나를 선택해서 실행하는 구문이에요. "분기"는 "갈림길"이라는 뜻입니다.

    예시로 확인¶

    #include <stdio.h>

int main(void) {
    int floor;

    printf("몇 층으로 가시겠습니까? (1~3): ");
    scanf("%d", &floor);

    switch (floor) {
        case 1:
            printf("1층: 로비입니다.\n");
            break;
        case 2:
            printf("2층: 식당입니다.\n");
            break;
        case 3:
            printf("3층: 옥상입니다.\n");
            break;
        default:
            printf("해당 층은 없습니다.\n");
            break;
    }

    return 0;
}

    실행 결과 (2를 입력했을 때): 

    몇 층으로 가시겠습니까? (1~3): 2
2층: 식당입니다.

    switch문의 각 부분을 하나씩 뜯어볼게요:

    • switch (floor) → 괄호 안의 변수 값을 확인합니다
    • case 1: → floor 값이 1이면 여기부터 실행합니다
    • break; → switch문을 빠져나갑니다 (아주 중요!)
    • default: → 어떤 case에도 해당하지 않을 때 실행합니다 (if문의 else와 같은 역할)

    💡 잠깐! switch의 괄호 안에는 정수형(int, char) 값만 넣을 수 있어요. 소수점이 있는 float이나 double은 넣을 수 없습니다. 문자(char)는 내부적으로 정수로 처리되므로 사용 가능해요!

    개념 2: break의 역할과 fall-through 🌊¶

    비유로 시작¶

    break는 엘리베이터의 문과 같습니다. 원하는 층에 도착하면 문이 열리고 내리죠? 만약 문이 고장 나서 안 열리면 어떻게 될까요? 엘리베이터가 그 다음 층, 그 다음 층... 계속 내려가 버립니다!

    break를 안 쓰면 바로 이런 일이 벌어져요. 해당 case를 실행한 뒤 멈추지 않고, 아래에 있는 case들을 연달아 실행합니다. 이것을 fall-through(폭포수 효과)라고 부릅니다.

    정확한 정의¶

    • break: switch문의 실행을 즉시 중단하고 switch 블록 밖으로 빠져나가는 키워드
    • fall-through: break가 없을 때 다음 case의 코드까지 연달아 실행되는 현상

    예시로 확인: break가 없으면?¶

    #include <stdio.h>

int main(void) {
    int choice = 2;

    printf("=== break가 없는 switch ===\n");

    switch (choice) {
        case 1:
            printf("1번 실행\n");
        case 2:
            printf("2번 실행\n");  // choice가 2이므로 여기부터 시작
        case 3:
            printf("3번 실행\n");  // break가 없어서 여기도 실행!
        case 4:
            printf("4번 실행\n");  // 여기도 실행!
        default:
            printf("default 실행\n");  // 여기도 실행!
    }

    return 0;
}

    실행 결과: 

    === break가 없는 switch ===
2번 실행
3번 실행
4번 실행
default 실행

    choice가 2인데 3번, 4번, default까지 모두 실행되었어요! 😱

    이번에는 break를 제대로 넣어볼게요:

    #include <stdio.h>

int main(void) {
    int choice = 2;

    printf("=== break가 있는 switch ===\n");

    switch (choice) {
        case 1:
            printf("1번 실행\n");
            break;
        case 2:
            printf("2번 실행\n");
            break;  // 여기서 switch를 빠져나감!
        case 3:
            printf("3번 실행\n");
            break;
        case 4:
            printf("4번 실행\n");
            break;
        default:
            printf("default 실행\n");
            break;
    }

    return 0;
}

    실행 결과: 

    === break가 있는 switch ===
2번 실행

    이제 2번만 깔끔하게 실행되었죠? ✨

    flowchart TD A["switch 시작: choice = 2"] --> B["case 1? → 아님, 건너뜀"] B --> C["case 2? → 일치!"] C --> D["'2번 실행' 출력"] D --> E{"break 있음?"} E -->|"break 있음 ✅"| F["switch 탈출!"] E -->|"break 없음 ❌"| G["case 3도 실행 (fall-through!)"] G --> H["case 4도 실행..."] H --> I["default도 실행..."]

    💡 잠깐! fall-through가 항상 나쁜 것만은 아니에요. 일부러 활용하는 경우도 있습니다. 예를 들어 여러 case에서 같은 동작을 해야 할 때요:

    case 1:
case 2:
case 3:
    printf("1~3 중 하나를 선택했습니다.\n");
    break;
    이렇게 하면 1, 2, 3 모두 같은 메시지를 출력합니다.

    개념 3: switch문 vs if-else if문 🤔¶

    비유로 시작¶

    if-else if는 목록을 위에서부터 쭉 훑어보는 것과 같아요. "이것인가? 아니면 저것인가?" 하나씩 대조합니다.

    switch는 목차에서 페이지를 찾아 바로 펴는 것과 같아요. "5번이구나? → 5번 페이지로 바로 이동!"

    둘 다 같은 결과를 만들 수 있지만, 상황에 따라 더 적합한 것이 다릅니다.

    정확한 정의¶

    flowchart LR subgraph IF["if-else if 방식"] A1["조건 1 확인"] --> A2["조건 2 확인"] A2 --> A3["조건 3 확인"] A3 --> A4["...순차 비교"] end subgraph SW["switch 방식"] B1["값 확인"] --> B2["해당 case로 점프"] end

    switch가 더 좋은 경우: - 하나의 변수를 여러 고정 값과 비교할 때 - 선택지가 3개 이상일 때 - 메뉴 선택처럼 정수/문자 값을 비교할 때

    if-else if가 더 좋은 경우: - 범위를 비교할 때 (예: score >= 90) - 여러 변수를 동시에 비교할 때 - 실수형(float/double) 값을 비교할 때 - 복잡한 논리 연산(&&, ||)이 필요할 때

    예시로 확인: 같은 기능, 다른 코드¶

    if-else if 버전:

    if (grade == 1) {
    printf("1학년입니다.\n");
} else if (grade == 2) {
    printf("2학년입니다.\n");
} else if (grade == 3) {
    printf("3학년입니다.\n");
} else if (grade == 4) {
    printf("4학년입니다.\n");
} else {
    printf("잘못된 학년입니다.\n");
}

    switch 버전:

    switch (grade) {
    case 1:
        printf("1학년입니다.\n");
        break;
    case 2:
        printf("2학년입니다.\n");
        break;
    case 3:
        printf("3학년입니다.\n");
        break;
    case 4:
        printf("4학년입니다.\n");
        break;
    default:
        printf("잘못된 학년입니다.\n");
        break;
}

    어떤가요? 선택지가 많을수록 switch문이 구조가 깔끔하고 한눈에 들어오죠? 각 case가 일정한 패턴으로 정렬되어 있어서 읽기 편합니다.

    🎯 기억하세요! switch문에서 case 뒤에 오는 값은 반드시 상수(고정된 값)여야 합니다. 변수나 계산식은 올 수 없어요! 

    case 1:       // ✅ 정수 상수 - OK
case 'A':     // ✅ 문자 상수 - OK
case x:       // ❌ 변수 - 컴파일 에러!
case x + 1:   // ❌ 계산식 - 컴파일 에러!

    개념 4: default와 문자(char)를 활용한 switch 🎨¶

    비유로 시작¶

    default는 자판기의 "해당 상품 없음" 안내와 같아요. 1번~5번 음료가 있는데 누군가 9번을 누르면? "해당 상품이 없습니다"라고 안내하죠. default가 바로 이 역할입니다.

    그리고 switch는 정수뿐 아니라 문자(char)도 사용할 수 있어요! 문자는 내부적으로 정수(ASCII 코드)로 저장되기 때문입니다.

    정확한 정의¶

    • default: 어떤 case에도 일치하지 않을 때 실행되는 부분입니다. 필수는 아니지만, 항상 넣어주는 습관이 좋아요.
    • char형 switch: 문자 하나를 비교할 때 case 'A': 형태로 사용합니다.

    예시로 확인¶

    #include <stdio.h>

int main(void) {
    char command;

    printf("명령어를 입력하세요 (a: 공격, d: 방어, h: 회복): ");
    scanf(" %c", &command);  // %c 앞의 공백에 주목!

    switch (command) {
        case 'a':
        case 'A':
            printf("⚔️ 공격합니다! 적에게 10 데미지!\n");
            break;
        case 'd':
        case 'D':
            printf("🛡️ 방어 자세를 취합니다!\n");
            break;
        case 'h':
        case 'H':
            printf("💚 체력을 20 회복합니다!\n");
            break;
        default:
            printf("❓ 알 수 없는 명령어입니다.\n");
            break;
    }

    return 0;
}

    여기서 두 가지 포인트를 짚어볼게요:

    포인트 1 — 대소문자 모두 처리하기: case 'a': 바로 다음에 case 'A':를 break 없이 이어 쓰면, 소문자 a든 대문자 A든 같은 동작을 합니다. 이것이 fall-through의 유용한 활용이에요!

    포인트 2 — scanf에서 " %c"의 공백: %c 앞에 공백을 하나 넣으면 이전에 남아있던 엔터(줄바꿈 문자)를 무시합니다. 이 공백이 없으면 엔터가 문자로 입력되어 버릴 수 있어요.

    💡 잠깐! 여기까지 핵심 개념 4가지를 모두 배웠어요! 정리하면: 1. switch문의 기본 구조 (switch - case - break - default) 2. break와 fall-through 3. switch vs if-else if 차이점 4. default와 char형 활용

    이제 실습으로 직접 손을 움직여 볼 시간입니다! 💪

    🔨 따라하기: 게임 메뉴 시스템 만들기¶

    🎯 실습 목표¶

    RPG 게임의 메인 메뉴를 만들어 봅니다. 사용자가 번호를 선택하면 해당 기능이 실행되고, '종료'를 선택할 때까지 메뉴가 반복됩니다.

    Step 1: 기본 메뉴 화면 출력하기¶

    먼저 메뉴 화면만 출력해 봅시다.

    #include <stdio.h>

int main(void) {
    printf("================================\n");
    printf("    🎮 용사의 모험 - 메인 메뉴\n");
    printf("================================\n");
    printf("    1. 새 게임\n");
    printf("    2. 이어하기\n");
    printf("    3. 캐릭터 정보\n");
    printf("    4. 종료\n");
    printf("================================\n");
    printf("    선택하세요 >> ");

    int choice;
    scanf("%d", &choice);

    printf("당신의 선택: %d\n", choice);

    return 0;
}

    ✅ 체크포인트: 실행하면 메뉴가 보이고, 숫자를 입력하면 "당신의 선택: X"가 출력되어야 해요!

    Step 2: switch문으로 선택 처리하기¶

    이제 입력받은 번호에 따라 다른 메시지를 출력하도록 switch문을 추가합니다.

    #include <stdio.h>

int main(void) {
    int choice;

    printf("================================\n");
    printf("    🎮 용사의 모험 - 메인 메뉴\n");
    printf("================================\n");
    printf("    1. 새 게임\n");
    printf("    2. 이어하기\n");
    printf("    3. 캐릭터 정보\n");
    printf("    4. 종료\n");
    printf("================================\n");
    printf("    선택하세요 >> ");
    scanf("%d", &choice);

    switch (choice) {
        case 1:
            printf("\n🌟 새로운 모험이 시작됩니다!\n");
            printf("용사여, 세계를 구하러 출발!\n");
            break;
        case 2:
            printf("\n📂 저장된 데이터를 불러옵니다...\n");
            printf("레벨 15 용사 데이터를 찾았습니다!\n");
            break;
        case 3:
            printf("\n👤 캐릭터 정보\n");
            printf("이름: 용사  레벨: 1  HP: 100\n");
            break;
        case 4:
            printf("\n👋 게임을 종료합니다. 안녕!\n");
            break;
        default:
            printf("\n❌ 1~4 사이의 숫자를 입력해주세요.\n");
            break;
    }

    return 0;
}

    ✅ 체크포인트: 1을 입력하면 "새로운 모험이 시작됩니다!", 5를 입력하면 "1~4 사이의 숫자를 입력해주세요."가 나와야 해요!

    Step 3: 반복문으로 메뉴 계속 보여주기¶

    지금은 한 번 선택하면 프로그램이 끝나죠? 실제 게임처럼 '종료'를 선택할 때까지 메뉴가 반복되도록 만들어 봅시다.

    #include <stdio.h>

int main(void) {
    int choice;
    int running = 1;  // 게임이 실행 중인지 표시 (1: 실행, 0: 종료)

    printf("🎮 용사의 모험에 오신 것을 환영합니다!\n\n");

    while (running) {
        // 메뉴 출력
        printf("================================\n");
        printf("         메 인  메 뉴\n");
        printf("================================\n");
        printf("    1. 새 게임\n");
        printf("    2. 이어하기\n");
        printf("    3. 캐릭터 정보\n");
        printf("    4. 종료\n");
        printf("================================\n");
        printf("    선택하세요 >> ");
        scanf("%d", &choice);

        switch (choice) {
            case 1:
                printf("\n🌟 새로운 모험이 시작됩니다!\n");
                printf("용사여, 세계를 구하러 출발!\n\n");
                break;
            case 2:
                printf("\n📂 저장된 데이터를 불러옵니다...\n");
                printf("레벨 15 용사 데이터를 찾았습니다!\n\n");
                break;
            case 3:
                printf("\n👤 캐릭터 정보\n");
                printf("이름: 용사  레벨: 1  HP: 100\n\n");
                break;
            case 4:
                printf("\n👋 게임을 종료합니다. 안녕!\n");
                running = 0;  // while 반복을 끝냄
                break;
            default:
                printf("\n❌ 1~4 사이의 숫자를 입력해주세요.\n\n");
                break;
        }
    }

    return 0;
}

    핵심은 running 변수예요. 4번(종료)을 선택하면 running = 0이 되어 while 반복이 끝납니다.

    ✅ 체크포인트: 1, 2, 3을 선택하면 메시지 출력 후 메뉴가 다시 나타나고, 4를 선택해야만 프로그램이 끝나야 해요!

    Step 4: 전투 메뉴 추가하기 (switch 안의 switch)¶

    "새 게임"을 선택하면 전투가 시작되도록 해볼까요? 전투에서도 메뉴가 필요하겠죠!

    #include <stdio.h>

int main(void) {
    int choice;
    int battleChoice;
    int running = 1;
    int hp = 100;
    int enemyHp = 50;

    printf("🎮 용사의 모험에 오신 것을 환영합니다!\n\n");

    while (running) {
        printf("================================\n");
        printf("         메 인  메 뉴\n");
        printf("================================\n");
        printf("    1. 전투 시작\n");
        printf("    2. 캐릭터 정보\n");
        printf("    3. 종료\n");
        printf("================================\n");
        printf("    선택하세요 >> ");
        scanf("%d", &choice);

        switch (choice) {
            case 1:
                printf("\n⚔️ 슬라임이 나타났다! (HP: %d)\n", enemyHp);
                printf("  a. 공격  b. 방어  c. 도망\n");
                printf("  행동 선택 >> ");

                char action;
                scanf(" %c", &action);  // 공백 잊지 마세요!

                switch (action) {
                    case 'a':
                        printf("  ⚔️ 용사의 공격! 슬라임에게 20 데미지!\n\n");
                        enemyHp -= 20;
                        break;
                    case 'b':
                        printf("  🛡️ 방어 자세! 데미지를 줄입니다.\n\n");
                        break;
                    case 'c':
                        printf("  🏃 도망쳤습니다!\n\n");
                        break;
                    default:
                        printf("  ❓ 잘못된 행동입니다.\n\n");
                        break;
                }
                break;

            case 2:
                printf("\n👤 용사 - HP: %d\n\n", hp);
                break;

            case 3:
                printf("\n👋 게임을 종료합니다.\n");
                running = 0;
                break;

            default:
                printf("\n❌ 1~3 사이의 숫자를 입력해주세요.\n\n");
                break;
        }
    }

    return 0;
}

    ✅ 체크포인트: 1을 선택하면 전투 메뉴가 나타나고, 'a'를 입력하면 공격 메시지가 출력되어야 해요! 2를 선택하면 현재 HP가 표시되어야 합니다.

    💡 잠깐! switch 안에 switch를 넣을 수도 있어요. 다만 코드가 길어지면 읽기 어려워질 수 있으니, 나중에 함수를 배
