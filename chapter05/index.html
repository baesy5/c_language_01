    Chapter 05: 컴퓨터에게 계산 시키기 — 연산자 활용¶

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • 산술 연산자(+, -, *, /, %)를 사용하여 계산 프로그램을 만들 수 있다
    • 정수 나눗셈과 나머지 연산의 차이를 이해하고 설명할 수 있다
    • 연산 우선순위와 괄호 사용법을 이해할 수 있다
    • 여러 변수와 연산자를 조합하여 의미 있는 결과를 출력할 수 있다

    예상 학습 시간: 약 1시간 30분

    📚 미리 알아야 할 것¶

    이전 장에서 우리는 변수라는 개념을 배웠어요. 기억나시나요?

    • 변수는 데이터를 담는 이름표 붙은 상자였죠
    • int, double, String 같은 자료형도 배웠어요
    • System.out.println()으로 화면에 결과를 출력하는 것도요
    • Scanner를 사용해 키보드로 값을 입력받는 방법도 살짝 맛봤어요

    이번 장에서는 그 상자(변수)에 담긴 숫자들을 가지고 계산을 해볼 거예요! 😄

    💡 왜 이걸 배우나요?¶

    컴퓨터(Computer)라는 이름 자체가 "계산하는 기계"라는 뜻이에요. 컴퓨터가 탄생한 가장 근본적인 이유가 바로 계산이죠.

    우리가 일상에서 당연하게 하는 계산들을 떠올려 봐요:

    • 🛒 마트에서 장보기: "사과 3개 × 1,500원 = 4,500원"
    • 💰 더치페이: "총 금액 45,000원 ÷ 3명 = 1인당 15,000원"
    • 🎮 게임 점수: "기본 점수 100 + 보너스 50 = 총 150점"
    • 📊 평균 계산: "(80 + 90 + 70) ÷ 3 = 80점"

    프로그래밍에서 연산자(Operator)는 이런 계산을 컴퓨터에게 시키는 명령 기호예요. 연산자를 알면, 컴퓨터를 나만의 초고속 계산기로 쓸 수 있습니다!

    🎯 이 장의 핵심 한 줄: 연산자는 컴퓨터에게 "이 숫자들로 이런 계산을 해!"라고 말하는 기호입니다.

    📚 핵심 개념¶

    개념 1: 산술 연산자 — 다섯 가지 계산 도구 🧮¶

    1. 비유로 시작¶

    여러분의 책상 위에 계산기가 있다고 상상해 보세요. 계산기에는 +, -, ×, ÷ 버튼이 있죠? Java에도 이와 똑같은 버튼들이 있어요. 다만 생김새가 약간 달라요. 키보드로 쉽게 입력할 수 있도록 기호가 조금 바뀐 거예요.

    2. 정확한 정의¶

    산술 연산자(Arithmetic Operator)란, 숫자 데이터에 대해 수학적 계산을 수행하는 기호입니다. Java에는 다섯 가지 기본 산술 연산자가 있어요.

    • + (덧셈): 두 수를 더합니다
    • - (뺄셈): 왼쪽 수에서 오른쪽 수를 뺍니다
    • * (곱셈): 두 수를 곱합니다 — 수학의 × 대신 별표(*)를 써요
    • / (나눗셈): 왼쪽 수를 오른쪽 수로 나눕니다 — 수학의 ÷ 대신 슬래시(/)를 써요
    • % (나머지): 나눈 후 나머지만 구합니다 — 이건 새로운 거예요!

    3. 예시로 확인¶

    int a = 10;
int b = 3;

System.out.println(a + b);   // 13   (덧셈)
System.out.println(a - b);   // 7    (뺄셈)
System.out.println(a * b);   // 30   (곱셈)
System.out.println(a / b);   // 3    (나눗셈 — 어라? 3.33...이 아니네?)
System.out.println(a % b);   // 1    (나머지 — 10 ÷ 3 = 3 나머지 1)

    💡 잠깐! 10 / 3의 결과가 3이라고요? 3.333...이 아니고요? 네, 맞아요! 이것이 바로 다음에 배울 정수 나눗셈의 비밀이에요.

    flowchart LR A["10"] --- PLUS["+"] --- B1["3"] --- R1["= 13"] C["10"] --- MINUS["-"] --- D1["3"] --- R2["= 7"] E["10"] --- MULT["*"] --- F1["3"] --- R3["= 30"] G["10"] --- DIV["/"] --- H1["3"] --- R4["= 3"] I["10"] --- MOD["%"] --- J1["3"] --- R5["= 1"]

    개념 2: 정수 나눗셈의 함정 — 소수점이 사라진다! 🕵️¶

    1. 비유로 시작¶

    피자 1판(8조각)을 3명이 공평하게 나눠 먹는다고 해봐요.

    • 한 사람당 2조각씩 먹을 수 있어요
    • 그러면 2조각이 남죠

    수학적으로는 8 ÷ 3 = 2.666...이지만, 피자를 실제로 나눌 때는 "한 사람당 2조각, 나머지 2조각"이라고 말하는 게 자연스럽죠. 정수 나눗셈이 바로 이 방식이에요!

    2. 정확한 정의¶

    정수(int)끼리 나누면, 결과도 정수입니다. 소수점 아래는 그냥 잘려 나가요(버림). 반올림이 아니라 버림이에요!

    이걸 정수 나눗셈(Integer Division)이라고 해요.

    3. 예시로 확인¶

    // 정수 ÷ 정수 = 정수 (소수점 버림!)
System.out.println(10 / 3);   // 3    (3.333... → 3)
System.out.println(7 / 2);    // 3    (3.5 → 3)
System.out.println(1 / 2);    // 0    (0.5 → 0)

// 소수점을 살리고 싶다면? double을 사용!
System.out.println(10.0 / 3);   // 3.3333333333333335
System.out.println(7.0 / 2);    // 3.5
System.out.println(1.0 / 2);    // 0.5

    핵심 규칙을 정리하면:

    • int / int → 결과는 int (소수점 사라짐)
    • double / int 또는 int / double → 결과는 double (소수점 살아남)

    🔑 꿀팁: 나눗셈에서 소수점 결과가 필요하다면, 숫자 하나에 .0만 붙여주세요! 10을 10.0으로 바꾸면 됩니다.

    개념 3: 나머지 연산자(%) — 숨은 영웅 🦸¶

    1. 비유로 시작¶

    초등학교 때 나눗셈을 처음 배울 때 기억나시나요?

    "13 나누기 4는 3 나머지 1"

    이때 /는 "몫"을 구하고, %는 "나머지"를 구하는 거예요. 즉, /와 %는 나눗셈의 짝꿍이에요!

    2. 정확한 정의¶

    나머지 연산자(%)는 왼쪽 수를 오른쪽 수로 나눈 후 나머지만 돌려주는 연산자입니다. 영어로는 modulo 또는 mod라고 불러요.

    3. 예시로 확인¶

    System.out.println(13 / 4);   // 3  (몫)
System.out.println(13 % 4);   // 1  (나머지)

System.out.println(10 % 5);   // 0  (딱 나눠떨어지면 나머지가 0)
System.out.println(10 % 3);   // 1
System.out.println(7 % 2);    // 1
System.out.println(8 % 2);    // 0

    🤔 "나머지 연산이 어디에 쓰일까요?" 정말 많이 쓰여요!

    • 짝수/홀수 판별: number % 2가 0이면 짝수, 1이면 홀수
    • 배수 확인: number % 3이 0이면 3의 배수
    • 시간 계산: 150분은 몇 시간 몇 분? → 150 / 60 = 2시간, 150 % 60 = 30분
    • 자릿수 추출: 1234 % 10 = 4 (일의 자리)
    // 짝수/홀수 판별
int number = 7;
System.out.println(number % 2);   // 1 → 홀수!

// 150분을 시간과 분으로 변환
int totalMinutes = 150;
int hours = totalMinutes / 60;    // 2시간
int minutes = totalMinutes % 60;  // 30분
System.out.println(hours + "시간 " + minutes + "분");  // 2시간 30분

    💡 잠깐! 여기까지 이해가 잘 되셨나요? /는 몫, %는 나머지 — 이 둘은 항상 짝꿍이라고 기억하세요! 어렵게 느껴져도 괜찮아요. 실습하면서 자연스럽게 익숙해질 거예요!

    개념 4: 연산 우선순위와 괄호 — 계산 순서의 비밀 📐¶

    1. 비유로 시작¶

    수학 시간에 배운 걸 떠올려 봐요:

    2 + 3 × 4 = ?

    답이 20인가요, 14인가요? 정답은 14예요! 곱셈이 덧셈보다 먼저 계산되니까요. Java도 수학과 똑같은 규칙을 따릅니다.

    2. 정확한 정의¶

    연산 우선순위(Operator Precedence)란, 여러 연산자가 한 줄에 있을 때 어떤 연산을 먼저 할지 정해놓은 규칙이에요.

    Java의 산술 연산 우선순위:

    • 1순위: () 괄호 안의 연산이 가장 먼저!
    • 2순위: *, /, % (곱셈, 나눗셈, 나머지)
    • 3순위: +, - (덧셈, 뺄셈)
    • 같은 순위끼리는 왼쪽에서 오른쪽 순서로 계산

    3. 예시로 확인¶

    // 우선순위 확인
System.out.println(2 + 3 * 4);       // 14  (곱셈 먼저: 2 + 12 = 14)
System.out.println((2 + 3) * 4);     // 20  (괄호 먼저: 5 * 4 = 20)

System.out.println(10 - 2 * 3);      // 4   (곱셈 먼저: 10 - 6 = 4)
System.out.println((10 - 2) * 3);    // 24  (괄호 먼저: 8 * 3 = 24)

// 같은 우선순위는 왼쪽 → 오른쪽
System.out.println(10 / 2 * 3);      // 15  (왼쪽부터: 5 * 3 = 15)
System.out.println(10 * 2 / 4);      // 5   (왼쪽부터: 20 / 4 = 5)
    flowchart TD EXPR["2 + 3 * 4"] STEP1["① 먼저: 3 * 4 = 12"] STEP2["② 다음: 2 + 12 = 14"] RESULT["결과: 14"] EXPR --> STEP1 --> STEP2 --> RESULT EXPR2["(2 + 3) * 4"] STEP3["① 괄호 먼저: 2 + 3 = 5"] STEP4["② 다음: 5 * 4 = 20"] RESULT2["결과: 20"] EXPR2 --> STEP3 --> STEP4 --> RESULT2

    🔑 꿀팁: 우선순위가 헷갈릴 때는 그냥 괄호를 쓰세요! 괄호는 항상 가장 먼저 계산되니까, 의도를 명확하게 표현할 수 있어요. 코드를 읽는 다른 사람(미래의 나 포함!)에게도 친절한 방법이에요.

    // 괄호를 사용하면 의도가 분명해져요
int price = 1000;
int quantity = 3;
int discount = 500;

// 의도: (가격 × 수량) - 할인
int total = (price * quantity) - discount;   // 2500 ✅ 명확!

    🔨 따라하기: 나만의 계산기 프로그램 만들기 🖩¶

    이제 배운 내용을 활용해서 두 숫자를 입력받고, 다섯 가지 연산 결과를 한 번에 보여주는 계산기를 만들어 볼게요!

    Step 1: 프로젝트 준비¶

    에디터(VS Code, IntelliJ 등)를 열고, Calculator.java라는 새 파일을 만들어 주세요.

    아래처럼 기본 틀을 작성합니다:

    import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        // 여기에 코드를 작성할 거예요!
    }
}

    ✅ 체크포인트: 파일 이름이 Calculator.java인지, public class Calculator와 이름이 같은지 확인하세요!

    Step 2: Scanner로 두 숫자 입력받기¶

    main 메서드 안에 다음 코드를 추가하세요:

    Scanner scanner = new Scanner(System.in);

System.out.println("===== 나만의 계산기 =====");

System.out.print("첫 번째 정수를 입력하세요: ");
int num1 = scanner.nextInt();

System.out.print("두 번째 정수를 입력하세요: ");
int num2 = scanner.nextInt();

    여기서 System.out.print()는 println()과 달리 줄바꿈 없이 출력해요. 그래서 사용자가 같은 줄에 숫자를 입력할 수 있어요.

    ✅ 체크포인트: 실행해서 숫자 두 개를 입력할 수 있는지 확인하세요!

    Step 3: 다섯 가지 연산 수행하기¶

    입력받은 코드 아래에 연산 결과를 변수에 저장하는 코드를 추가합니다:

    // 다섯 가지 연산 수행
int sum = num1 + num2;          // 덧셈
int difference = num1 - num2;   // 뺄셈
int product = num1 * num2;      // 곱셈
int quotient = num1 / num2;     // 몫 (정수 나눗셈)
int remainder = num1 % num2;    // 나머지

    💡 포인트: 연산 결과를 바로 출력해도 되지만, 변수에 저장해두면 나중에 여러 번 사용할 수 있어서 편리해요!

    Step 4: 결과 출력하기¶

    이제 계산 결과를 예쁘게 출력해 봐요:

    // 결과 출력
System.out.println();  // 빈 줄 하나
System.out.println("===== 계산 결과 =====");
System.out.println(num1 + " + " + num2 + " = " + sum);
System.out.println(num1 + " - " + num2 + " = " + difference);
System.out.println(num1 + " * " + num2 + " = " + product);
System.out.println(num1 + " / " + num2 + " = " + quotient + " (몫)");
System.out.println(num1 + " % " + num2 + " = " + remainder + " (나머지)");

    ✅ 체크포인트: 10과 3을 입력해 보세요. 결과가 아래와 같이 나와야 해요: 

    ===== 계산 결과 =====
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3 (몫)
10 % 3 = 1 (나머지)

    Step 5: 소수점 나눗셈 결과도 추가하기¶

    정수 나눗셈만으로는 아쉬우니, 실수 나눗셈 결과도 함께 보여주면 어떨까요?

    Step 3의 연산 코드 아래에 한 줄을 추가하세요:

    double realDivision = (double) num1 / num2;  // 실수 나눗셈

    여기서 (double)은 형변환(casting)이라고 해요. num1을 계산할 때 잠깐 double로 바꿔서, 소수점 결과를 얻을 수 있게 하는 거예요. 마치 "이 정수를 실수처럼 취급해줘!"라고 말하는 것과 같아요.

    그리고 출력 부분에 한 줄을 추가합니다:

    System.out.println(num1 + " ÷ " + num2 + " = " + realDivision + " (실수 나눗셈)");

    ✅ 체크포인트: 다시 10과 3을 입력하면, 10 ÷ 3 = 3.3333333333333335 (실수 나눗셈) 이 추가로 나타나야 해요!

    Step 6: 보너스 — 시간 변환 기능 추가 ⏰¶

    나머지 연산의 실전 활용을 경험해 봐요! 계산기 코드 맨 아래에 다음을 추가하세요:

    // 보너스: 시간 변환
System.out.println();
System.out.print("변환할 분(minute)을 입력하세요: ");
int totalMinutes = scanner.nextInt();

int hours = totalMinutes / 60;     // 몫 = 시간
int mins = totalMinutes % 60;      // 나머지 = 분

System.out.println(totalMinutes + "분 = " + hours + "시간 " + mins + "분");

scanner.close();

    ✅ 체크포인트: 135를 입력하면 135분 = 2시간 15분이 나와야 해요!

    💡 여기까지 정말 잘 따라오셨어요! 🎉 혹시 에러가 나더라도 당황하지 마세요. 오타를 꼼꼼히 확인해 보면 대부분 해결됩니다!

    📝 전체 코드¶

    아래는 완성된 전체 코드예요. 복사해서 바로 실행할 수 있습니다!

    import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // === 계산기 ===
        System.out.println("===== 나만의 계산기 =====");

        System.out.print("첫 번째 정수를 입력하세요: ");
        int num1 = scanner.nextInt();

        System.out.print("두 번째 정수를 입력하세요: ");
        int num2 = scanner.nextInt();

        // 다섯 가지 연산 수행
        int sum = num1 + num2;
        int difference = num1 - num2;
        int product = num1 * num2;
        int quotient = num1 / num2;
        int remainder = num1 % num2;
        double realDivision = (double) num1 / num2;

        // 결과 출력
        System.out.println();
        System.out.println("===== 계산 결과 =====");
        System.out.println(num1 + " + " + num2 + " = " + sum);
        System.out.println(num1 + " - " + num2 + " = " + difference);
        System.out.println(num1 + " * " + num2 + " = " + product);
        System.out.println(num1 + " / " + num2 + " = " + quotient + " (몫)");
        System.out.println(num1 + " % " + num2 + " = " + remainder + " (나머지)");
        System.out.println(num1 + " ÷ " + num2 + " = " + realDivision + " (실수 나눗셈)");

        // === 보너스: 시간 변환 ===
        System.out.println();
        System.out.print("변환할 분(minute)을 입력하세요: ");
        int totalMinutes = scanner.nextInt();

        int hours = totalMinutes / 60;
        int mins = totalMinutes % 60;

        System.out.println(totalMinutes + "분 = " + hours + "시간 " + mins + "분");

        scanner.close();
    }
}

    ⚠️ 자주 하는 실수¶

    실수 1: 정수 나눗셈 결과가 이상해요! 😱¶

    int result = 5 / 2;
System.out.println(result);  // 2 (2.5가 아님!)
    • 왜 이런 일이 생기나요?: int / int는 결과도 int예요. 소수점은 반올림이 아니라 버림 처리됩니다.
    • 이렇게 해결해요: 소수점 결과가 필요하면, 하나라도 double로 만들어 주세요!
    double result = 5.0 / 2;    // 2.5 ✅
// 또는
double result = (double) 5 / 2;  // 2.5 ✅
