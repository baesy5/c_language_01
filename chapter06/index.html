    갈림길에서 선택하기 — if 조건문¶

    Part 2: [1권] 프로그램에 판단력 불어넣기

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • if, else if, else 구조를 이해하고 조건에 따라 다른 결과를 출력할 수 있다
    • 비교 연산자(==, !=, <, >, <=, >=)를 활용한 조건식을 작성할 수 있다
    • 논리 연산자(&&, ||, !)를 사용하여 복합 조건을 만들 수 있다
    • '성적 등급 판별기' 프로그램을 직접 만들 수 있다

    예상 학습 시간: 약 2시간

    📚 미리 알아야 할 것¶

    이전 장까지 우리는 이런 것들을 배웠어요:

    • 변수: 값을 담아두는 상자 (int age = 20;)
    • 자료형: 정수(int), 실수(double), 문자열(String) 등
    • Scanner: 사용자로부터 입력을 받는 방법
    • 출력: System.out.println()으로 화면에 결과 보여주기

    이 개념들이 기억나지 않더라도 괜찮아요. 코드를 따라 치면서 자연스럽게 복습될 거예요! 😊

    💡 왜 이걸 배우나요?¶

    지금까지 우리가 만든 프로그램은 위에서 아래로 한 줄씩 순서대로 실행되기만 했어요. 마치 직선 도로를 따라가는 것처럼요.

    하지만 현실은 어떤가요?

    • "비가 오면 우산을 챙기고, 아니면 그냥 나간다"
    • "잔액이 충분하면 결제하고, 아니면 잔액 부족 알림을 띄운다"
    • "나이가 19세 이상이면 입장 가능, 아니면 입장 불가"

    이렇게 조건에 따라 다른 행동을 하는 것이 바로 프로그래밍의 핵심 능력입니다. 이것을 가능하게 해주는 것이 바로 if 조건문이에요.

    if문을 배우는 순간, 여러분의 프로그램은 "판단"할 줄 아는 똑똑한 프로그램이 됩니다! 🧠

    📚 핵심 개념¶

    개념 1: if문 — "만약 ~라면"¶

    🎨 비유로 시작해요

    신호등을 떠올려 보세요.

    • 🟢 초록불이면 → 건넌다
    • 🔴 빨간불이면 → 멈춘다

    우리는 매일 이렇게 "조건을 확인하고, 그에 따라 행동"합니다. Java의 if문이 바로 이 역할을 해요.

    📖 정확한 정의

    if문은 괄호 안의 조건이 참(true)일 때만 중괄호 안의 코드를 실행하는 구문입니다.

    if (조건) {
    // 조건이 참일 때 실행되는 코드
}

    🔍 예시로 확인

    int temperature = 35;

if (temperature >= 30) {
    System.out.println("오늘은 정말 더운 날이에요! 🥵");
}

    temperature가 35이고, 35는 30 이상이므로 조건이 참(true)입니다. 따라서 "오늘은 정말 더운 날이에요! 🥵"가 출력됩니다.

    만약 temperature가 25라면? 조건이 거짓(false)이므로 중괄호 안의 코드는 건너뛰고 다음으로 넘어갑니다.

    개념 2: if-else — "만약 ~라면, 그렇지 않으면"¶

    🎨 비유로 시작해요

    아까 신호등 비유를 더 확장해 볼게요:

    • 초록불이면 → 건넌다
    • 그렇지 않으면 → 멈춘다

    "그렇지 않으면"에 해당하는 부분이 바로 else입니다.

    📖 정확한 정의

    else는 if의 조건이 거짓(false)일 때 실행되는 코드를 지정합니다.

    if (조건) {
    // 조건이 참일 때
} else {
    // 조건이 거짓일 때
}

    🔍 예시로 확인

    int age = 16;

if (age >= 19) {
    System.out.println("입장 가능합니다. 🎬");
} else {
    System.out.println("미성년자는 입장할 수 없습니다. 🚫");
}

    age가 16이므로 age >= 19는 거짓입니다. 따라서 else 블록이 실행되어 "미성년자는 입장할 수 없습니다. 🚫"가 출력됩니다.

    아래 다이어그램으로 흐름을 확인해 보세요:

    flowchart TD A["age = 16"] --> B{"age >= 19?"} B -- "참(true)" --> C["입장 가능합니다 🎬"] B -- "거짓(false)" --> D["미성년자는 입장할 수 없습니다 🚫"] C --> E["프로그램 계속 진행"] D --> E

    💡 잠깐! 여기까지 이해가 잘 되셨나요? if는 "만약 ~라면", else는 "그렇지 않으면"이라고 기억하면 쉬워요!

    개념 3: else if — 여러 갈림길¶

    🎨 비유로 시작해요

    이번엔 신호등을 세 가지 색으로 확장해 볼게요:

    • 🟢 초록불이면 → 건넌다
    • 🟡 노란불이면 → 주의한다
    • 🔴 빨간불이면 → 멈춘다

    선택지가 2개가 아니라 3개 이상일 때, else if를 사용합니다.

    📖 정확한 정의

    else if는 이전 조건이 거짓일 때 추가로 다른 조건을 확인하는 구문입니다. 원하는 만큼 여러 개를 연결할 수 있어요.

    if (조건1) {
    // 조건1이 참일 때
} else if (조건2) {
    // 조건1은 거짓이고, 조건2가 참일 때
} else if (조건3) {
    // 조건1, 2 모두 거짓이고, 조건3이 참일 때
} else {
    // 위 조건 모두 거짓일 때
}

    🔍 예시로 확인

    int score = 85;

if (score >= 90) {
    System.out.println("A등급입니다! 🌟");
} else if (score >= 80) {
    System.out.println("B등급입니다! 👍");
} else if (score >= 70) {
    System.out.println("C등급입니다.");
} else {
    System.out.println("더 노력이 필요해요. 💪");
}

    score가 85일 때, 첫 번째 조건 score >= 90은 거짓입니다. 그래서 다음으로 넘어가 score >= 80을 확인하는데, 이건 참이에요! 따라서 "B등급입니다! 👍"이 출력됩니다.

    🔑 핵심 포인트: else if는 위에서부터 순서대로 확인합니다. 하나라도 참이면 그 블록을 실행하고 나머지는 모두 건너뜁니다.

    flowchart TD A["score = 85"] --> B{"score >= 90?"} B -- "참" --> C["A등급 🌟"] B -- "거짓" --> D{"score >= 80?"} D -- "참" --> E["B등급 👍"] D -- "거짓" --> F{"score >= 70?"} F -- "참" --> G["C등급"] F -- "거짓" --> H["더 노력이 필요해요 💪"]

    개념 4: 비교 연산자와 논리 연산자¶

    조건문의 괄호 안에는 조건식이 들어갑니다. 조건식을 만드는 도구가 바로 비교 연산자와 논리 연산자예요.

    비교 연산자 — "두 값을 비교한다"¶

    • == : 같다 (a == b → a와 b가 같은가?)
    • != : 같지 않다 (a != b → a와 b가 다른가?)
    • > : 크다 (a > b → a가 b보다 큰가?)
    • < : 작다 (a < b → a가 b보다 작은가?)
    • >= : 크거나 같다
    • <= : 작거나 같다
    int x = 10;
System.out.println(x == 10);  // true
System.out.println(x != 5);   // true
System.out.println(x > 20);   // false
System.out.println(x <= 10);  // true

    논리 연산자 — "조건을 조합한다"¶

    때로는 조건 하나로는 부족할 때가 있어요. 예를 들어 "나이가 19세 이상이고 티켓이 있으면 입장 가능"처럼요.

    • && (그리고/AND) : 두 조건이 모두 참이어야 참
    • || (또는/OR) : 두 조건 중 하나만 참이어도 참
    • ! (부정/NOT) : 참을 거짓으로, 거짓을 참으로 뒤집기
    int age = 20;
boolean hasTicket = true;

// AND: 둘 다 참이어야 참
if (age >= 19 && hasTicket) {
    System.out.println("입장하세요! 🎉");
}

// OR: 하나만 참이어도 참
boolean isWeekend = false;
boolean isHoliday = true;
if (isWeekend || isHoliday) {
    System.out.println("오늘은 쉬는 날이에요! 😄");
}

// NOT: 뒤집기
boolean isRaining = false;
if (!isRaining) {
    System.out.println("우산 없이 나가도 돼요! ☀️");
}
    flowchart LR subgraph "AND (&&)" A1["조건1: 참"] --- A2["조건2: 참"] --- A3["결과: ✅ 참"] end subgraph "OR (||)" B1["조건1: 거짓"] --- B2["조건2: 참"] --- B3["결과: ✅ 참"] end subgraph "NOT (!)" C1["조건: 거짓"] --- C2["결과: ✅ 참"] end

    💡 잠깐! 여기까지 오셨다면 핵심 개념은 모두 배운 거예요! 이제 직접 코드를 작성하면서 손에 익혀 볼 시간입니다. 🎯

    🔨 따라하기: 성적 등급 판별기 만들기¶

    사용자로부터 점수를 입력받아 A~F 등급을 알려주는 프로그램을 단계별로 만들어 봅시다!

    Step 1: 프로젝트 준비¶

    새 Java 파일을 만들어 주세요. 파일 이름은 GradeChecker.java로 해볼게요.

    import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        System.out.println("=== 성적 등급 판별기 ===");
    }
}

    ✅ 체크포인트: 실행하면 === 성적 등급 판별기 ===가 출력되어야 해요!

    Step 2: 점수 입력받기¶

    Scanner를 사용해서 사용자에게 점수를 입력받아 봅시다.

    import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 성적 등급 판별기 ===");
        System.out.print("점수를 입력하세요 (0~100): ");
        int score = scanner.nextInt();

        System.out.println("입력한 점수: " + score);
    }
}

    ✅ 체크포인트: 실행 후 85를 입력하면 입력한 점수: 85가 나와야 해요!

    Step 3: if-else if-else로 등급 판별하기¶

    이제 핵심입니다! 점수에 따라 등급을 나눠 볼게요.

    import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 성적 등급 판별기 ===");
        System.out.print("점수를 입력하세요 (0~100): ");
        int score = scanner.nextInt();

        if (score >= 90) {
            System.out.println("🌟 A등급! 훌륭합니다!");
        } else if (score >= 80) {
            System.out.println("👍 B등급! 잘했어요!");
        } else if (score >= 70) {
            System.out.println("😊 C등급! 괜찮아요!");
        } else if (score >= 60) {
            System.out.println("😐 D등급. 조금 더 노력해요.");
        } else {
            System.out.println("😢 F등급. 다음엔 꼭 잘할 수 있어요!");
        }
    }
}

    ✅ 체크포인트: 여러 점수를 넣어보세요! - 95 입력 → A등급 - 82 입력 → B등급 - 55 입력 → F등급

    Step 4: 잘못된 입력 걸러내기 (복합 조건 활용!)¶

    점수가 0~100 범위를 벗어나면 어떡할까요? && 연산자로 범위를 검사해 봅시다.

    import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 성적 등급 판별기 ===");
        System.out.print("점수를 입력하세요 (0~100): ");
        int score = scanner.nextInt();

        if (score < 0 || score > 100) {
            System.out.println("⚠️ 0~100 사이의 점수를 입력해주세요!");
        } else if (score >= 90) {
            System.out.println("🌟 A등급! 훌륭합니다!");
        } else if (score >= 80) {
            System.out.println("👍 B등급! 잘했어요!");
        } else if (score >= 70) {
            System.out.println("😊 C등급! 괜찮아요!");
        } else if (score >= 60) {
            System.out.println("😐 D등급. 조금 더 노력해요.");
        } else {
            System.out.println("😢 F등급. 다음엔 꼭 잘할 수 있어요!");
        }
    }
}

    score < 0 || score > 100은 "점수가 0보다 작거나 또는 100보다 크면"이라는 뜻이에요. 이 조건을 가장 먼저 확인해서 잘못된 입력을 걸러냅니다.

    ✅ 체크포인트: -5나 150을 입력했을 때 경고 메시지가 나오면 성공!

    Step 5: 추가 메시지 — 논리 연산자 활용¶

    90점 이상이면서 100점인 경우 특별한 메시지를 추가해 봅시다.

    등급 판별 코드의 첫 번째 조건 부분을 이렇게 바꿔보세요:

    if (score < 0 || score > 100) {
    System.out.println("⚠️ 0~100 사이의 점수를 입력해주세요!");
} else if (score == 100) {
    System.out.println("🏆 만점! 완벽합니다!");
} else if (score >= 90) {
    System.out.println("🌟 A등급! 훌륭합니다!");
}
// ... 나머지 else if, else는 그대로

    여기서 ==는 "정확히 같다"라는 뜻의 비교 연산자입니다. score == 100은 "점수가 정확히 100인가?"를 확인해요.

    ✅ 체크포인트: 100을 입력하면 "🏆 만점! 완벽합니다!"가 나와야 해요!

    📝 전체 코드¶

    지금까지 완성한 코드를 한눈에 볼게요. 복사해서 바로 실행할 수 있습니다!

    import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 성적 등급 판별기 ===");
        System.out.print("점수를 입력하세요 (0~100): ");

        else if (score >= 90) {

    ⚠️ 자주 하는 실수¶

    실수 1: =와 == 혼동하기 🚨¶

    이것은 입문자가 가장 많이 하는 실수입니다!

    • = → 대입 연산자 (값을 넣는 것)
    • == → 비교 연산자 (같은지 확인하는 것)
    // ❌ 잘못된 코드 — 대입을 해버림!
if (score = 100) {   // 컴파일 에러 발생!
    System.out.println("만점!");
}

// ✅ 올바른 코드 — 비교를 함
if (score == 100) {
    System.out.println("만점!");
}

    왜 이런 일이 생기나요? 수학에서는 =가 "같다"의 의미이지만, 프로그래밍에서 =는 "오른쪽 값을 왼쪽에 넣어라"는 뜻이에요. "같은지 비교"하려면 반드시 ==를 써야 합니다.

    이렇게 기억하세요: 비교할 때는 눈이 두 개 👀 → ==

    실수 2: 중괄호 {} 빠뜨리기¶

    // ❌ 위험한 코드
if (score >= 90)
    System.out.println("A등급입니다!");
    System.out.println("축하합니다!");  // 이 줄은 if와 무관하게 항상 실행됨!

// ✅ 안전한 코드
if (score >= 90) {
    System.out.println("A등급입니다!");
    System.out.println("축하합니다!");
}

    왜 이런 일이 생기나요? 중괄호가 없으면 if문은 바로 다음 한 줄만 자기 것으로 인식합니다. 두 번째 줄의 "축하합니다!"는 점수에 관계없이 항상 출력되어 버려요.

    이렇게 해결해요: 습관적으로 if 뒤에 항상 중괄호를 쓰세요. 실행할 코드가 한 줄이더라도 중괄호를 넣는 것이 안전합니다.

    실수 3: else if 순서를 잘못 배치하기¶

    // ❌ 잘못된 순서
if (score >= 60) {
    System.out.println("D등급");
} else if (score >= 90) {      // 여기에 절대 도달하지 못함!
    System.out.println("A등급");
}

    왜 이런 일이 생기나요? 95점을 넣어도 첫 번째 조건 score >= 60이 이미 참이므로 "D등급"이 출력됩니다. 90점 이상 확인은 영원히 실행되지 않아요!

    이렇게 해결해요: 더 엄격한 조건(큰 숫자)부터 먼저 검사하세요. 90 → 80 → 70 → 60 순서로 내려오는 것이 올바릅니다.
