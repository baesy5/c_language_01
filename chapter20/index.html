    Chapter 20: 축하해, 게임 개발자! — 다음 단계로 나아가기 🎉¶

    🎯 이 장의 학습 목표¶

    • 자신이 만든 게임을 개선할 아이디어를 스스로 구상할 수 있다

    📚 미리 알아야 할 것¶

    이 장은 20차시의 마지막입니다. 지금까지 배운 모든 것을 돌아보는 시간이에요. 특별히 새로운 문법을 배우지는 않으니, 편안한 마음으로 읽어주세요. 😊

    🏆 먼저, 자신을 축하해주세요!¶

    2주 전의 자신을 떠올려 보세요.

    #include <stdio.h>가 뭔지도 몰랐던 때, printf를 처음 써보고 화면에 글자가 나왔을 때의 그 신기함을 기억하시나요?

    그런데 지금은요?

    • ✅ 숫자 맞추기 게임을 직접 만들었습니다
    • ✅ 텍스트 어드벤처 게임을 완성했습니다
    • ✅ 변수, 조건문, 반복문, 함수, 배열을 모두 사용할 줄 압니다

    당신은 이미 게임 개발자입니다. 작든 크든, 자기 손으로 동작하는 프로그램을 만든 사람은 모두 개발자예요. 🎮

    💡 잠깐! "겨우 이 정도로?"라는 생각이 드시나요? 모든 프로 개발자도 처음에는 Hello, World!부터 시작했어요. 첫 걸음을 뗀 것 자체가 대단한 일입니다!

    📚 핵심 개념 1: 2주간의 여정 총정리¶

    비유로 시작¶

    프로그래밍을 배우는 건 마치 레고 블록을 하나씩 모으는 것과 같아요. 처음엔 블록 하나하나가 뭔지 배우고, 나중엔 그 블록들을 조합해서 멋진 작품을 만들죠.

    여러분이 2주간 모은 레고 블록들을 한눈에 정리해 볼게요.

    graph TD A["🧱 1주차: 블록 모으기"] --> B["변수 — 데이터를 담는 상자"] A --> C["조건문 — 갈림길에서 선택하기"] A --> D["반복문 — 같은 일을 여러 번"] A --> E["함수 — 코드를 묶어서 재사용"] A --> F["배열 — 상자를 한 줄로 정리"] G["🎮 2주차: 작품 만들기"] --> H["숫자 맞추기 게임"] G --> I["텍스트 어드벤처 게임"] B --> G C --> G D --> G E --> G F --> G

    각 블록이 게임에서 어떻게 쓰였는지 확인해 볼까요?¶

    🧱 변수 — 플레이어의 체력, 정답 숫자, 시도 횟수를 저장했어요 

    int hp = 100;
int answer = 42;
int tries = 0;

    🧱 조건문 — 플레이어의 선택에 따라 다른 결과를 보여줬어요 

    if (choice == 1) {
    printf("동굴에 들어갑니다!\n");
} else {
    printf("숲으로 향합니다!\n");
}

    🧱 반복문 — 게임이 끝날 때까지 계속 진행했어요 

    while (hp > 0) {
    // 게임 계속 진행
}

    🧱 함수 — 전투, 탐험 같은 기능을 깔끔하게 분리했어요 

    void battle(int *hp) {
    printf("전투 시작!\n");
    // 전투 로직
}

    🧱 배열 — 여러 데이터를 한 번에 관리했어요 

    char name[20] = "용사";
int scores[5] = {100, 95, 88, 72, 60};

    💡 잠깐! 이 다섯 가지 개념만 잘 조합해도 놀라울 정도로 다양한 프로그램을 만들 수 있어요. 여러분은 이미 그 기초를 탄탄히 쌓은 거예요!

    📚 핵심 개념 2: 게임을 더 멋지게 — 개선 아이디어¶

    비유로 시작¶

    집을 지었다고 끝이 아니죠? 가구를 배치하고, 벽지를 바꾸고, 정원도 꾸미잖아요. 게임도 마찬가지예요!

    여러분이 만든 게임에 바로 추가할 수 있는 아이디어 세 가지를 소개합니다.

    아이디어 ① 난이도 조절 기능¶

    게임 시작 전에 난이도를 선택하게 만들어 보세요.

    printf("난이도를 선택하세요 (1:쉬움 2:보통 3:어려움): ");
scanf("%d", &difficulty);

if (difficulty == 1) {
    max_number = 10;   // 1~10 사이 숫자
    max_tries = 10;
} else if (difficulty == 2) {
    max_number = 50;   // 1~50 사이 숫자
    max_tries = 7;
} else {
    max_number = 100;  // 1~100 사이 숫자
    max_tries = 5;
}

    아이디어 ② 최고 기록 저장¶

    배열을 활용해서 최고 점수를 저장해 보세요.

    int high_scores[3] = {999, 999, 999}; // 적은 시도 = 높은 점수

// 게임 끝난 후
if (tries < high_scores[2]) {
    high_scores[2] = tries;
    printf("🏆 새로운 기록! %d번 만에 맞췄습니다!\n", tries);
}

    아이디어 ③ 텍스트 어드벤처에 새로운 스테이지 추가¶

    함수를 하나 더 만들면 새로운 모험이 추가돼요.

    void secret_stage(int *hp) {
    printf("\n🗝️ === 비밀의 방 ===\n");
    printf("어두운 방에 보물상자가 있습니다.\n");
    printf("1. 상자를 연다  2. 그냥 지나간다\n");
    // 여러분의 상상력으로 채워보세요!
}

    📚 핵심 개념 3: 다음 단계로 나아가기¶

    비유로 시작¶

    여러분은 지금 1층 건물을 완성한 거예요. C언어에는 2층, 3층, 그 위로도 올라갈 수 있는 멋진 주제들이 기다리고 있어요.

    graph BT A["🏠 1층: 지금 여러분의 위치\n변수, 조건문, 반복문, 함수, 배열"] B["🏗️ 2층: 다음 단계\n포인터, 구조체, 파일 입출력"] C["🏰 3층: 심화\n동적 메모리, 자료구조, 알고리즘"] A --> B B --> C

    다음에 배우면 좋을 주제 세 가지¶

    📌 포인터 (Pointer) - 메모리 주소를 직접 다루는 기술이에요 - C언어의 꽃이라고 불릴 만큼 중요하고, 처음엔 어렵게 느껴지지만 이해하면 프로그래밍 실력이 한 단계 올라가요

    📌 구조체 (Struct) - 관련 있는 변수들을 하나로 묶는 기능이에요 - 게임 캐릭터의 이름, 체력, 공격력을 하나의 "캐릭터" 타입으로 만들 수 있어요

    📌 파일 입출력 (File I/O) - 게임 기록을 파일에 저장하고 불러오는 기능이에요 - 프로그램을 껐다 켜도 데이터가 사라지지 않아요

    📝 전체 여정 요약 코드¶

    2주간 배운 모든 개념이 담긴 미니 코드예요. 마지막으로 한 번 실행해 보세요!

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

// 함수 선언
void congratulate(char name[], int days);

int main(void) {
    // 변수
    char name[20];
    int concepts[5] = {1, 2, 3, 4, 5};
    char *topic[] = {"변수", "조건문", "반복문", "함수", "배열"};
    int i;
    int days = 20;

    printf("🎓 === C언어 학습 수료증 === 🎓\n\n");
    printf("이름을 입력하세요: ");
    scanf("%s", name);

    // 반복문 + 배열
    printf("\n📋 마스터한 개념 목록:\n");
    for (i = 0; i < 5; i++) {
        printf("  ✅ %d. %s\n", concepts[i], topic[i]);
    }

    // 조건문
    if (days == 20) {
        printf("\n🏆 20차시 완주 성공!\n");
    }

    // 함수 호출
    congratulate(name, days);

    return 0;
}
