    📦 데이터를 담는 상자 — 변수와 자료형¶

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • 변수가 무엇인지 이해하고 int, float, char 자료형을 구분할 수 있다
    • 변수를 선언하고 값을 저장·변경하는 코드를 작성할 수 있다
    • scanf를 사용하여 사용자로부터 값을 입력받을 수 있다
    • 입력받은 값으로 간단한 계산을 수행하는 프로그램을 만들 수 있다

    ⏱️ 예상 학습 시간: 약 2시간

    📚 미리 알아야 할 것¶

    지난 장에서 우리는 printf를 사용해 화면에 글자를 출력하는 방법을 배웠어요. "Hello, World!"를 출력하며 C 프로그램의 기본 구조도 익혔죠.

    #include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}

    이 구조가 기억나시죠? 오늘은 여기에 데이터를 저장하고 활용하는 능력을 더해볼 거예요! 🎉

    💡 왜 이걸 배우나요?¶

    지금까지 우리가 만든 프로그램은 항상 같은 말만 했어요. "Hello, World!"를 출력하라고 하면 언제나 똑같은 문장만 보여줬죠.

    하지만 진짜 프로그램은 다릅니다. 계산기는 사용자가 입력한 숫자에 따라 다른 결과를 보여주고, 게임은 플레이어의 점수를 기억하고 업데이트해요.

    이런 일을 하려면 데이터를 어딘가에 저장해둘 수 있어야 합니다. 그 "어딘가"가 바로 오늘 배울 변수예요!

    📚 핵심 개념¶

    개념 1: 변수 — 이름표가 붙은 상자 📦¶

    비유로 시작해 볼게요.

    여러분의 책상 위에 상자가 하나 있다고 상상해 보세요. 이 상자에는 "나이"라는 이름표가 붙어 있고, 안에는 숫자 25가 들어 있어요. 나중에 생일이 지나면 25를 꺼내고 26을 넣을 수도 있겠죠.

    이것이 바로 변수(variable)입니다!

    정확한 정의:

    🔑 변수란 데이터를 저장해두는 메모리 공간에 이름을 붙인 것입니다. 이름을 통해 값을 저장하거나, 저장된 값을 꺼내 쓸 수 있어요.

    예시로 확인해 볼까요?

    int age = 25;

    이 한 줄이 하는 일을 풀어서 설명하면:

    • int → 이 상자는 정수(整數)를 담는 상자야!
    • age → 이 상자의 이름은 age야!
    • = 25 → 이 상자에 25를 넣어줘!
    flowchart LR A["int age = 25;"] --> B["자료형: int\n(정수 전용 상자)"] A --> C["변수 이름: age\n(상자의 이름표)"] A --> D["값: 25\n(상자 안에 든 것)"]

    💡 잠깐! "변수"라는 이름에서 힌트를 얻을 수 있어요. 변할 수 있는 수, 즉 안에 든 값이 바뀔 수 있다는 뜻이에요!

    개념 2: 자료형 — 상자의 종류 🏷️¶

    비유로 시작합니다.

    택배 상자에도 종류가 있죠? 책을 담는 상자, 과일을 담는 상자, 깨지기 쉬운 물건을 담는 상자… 각각 크기와 용도가 다릅니다.

    프로그래밍에서도 마찬가지예요. 담으려는 데이터의 종류에 따라 상자의 타입이 달라집니다. 이것을 자료형(data type)이라고 해요.

    정확한 정의:

    🔑 자료형이란 변수에 저장할 데이터의 종류와 크기를 지정하는 것입니다.

    오늘 배울 세 가지 자료형을 소개할게요:

    int — 정수를 담는 상자 - 소수점이 없는 숫자를 담아요 - 예: 1, 42, -7, 0, 1000

    float — 소수점 숫자를 담는 상자 - 소수점이 있는 숫자를 담아요 - 예: 3.14, 98.6, -0.5, 100.0

    char — 글자 하나를 담는 작은 상자 - 딱 한 글자만 담을 수 있어요 - 예: 'A', 'z', '7', '!' - ⚠️ 반드시 작은따옴표(' ')로 감싸요

    flowchart TB subgraph INT ["🔢 int (정수)"] I1["42"] I2["-7"] I3["0"] end subgraph FLOAT ["🔣 float (소수점)"] F1["3.14"] F2["98.6"] F3["-0.5"] end subgraph CHAR ["🔤 char (글자 하나)"] C1["'A'"] C2["'z'"] C3["'!'"] end

    예시 코드:

    int count = 10;          // 정수 10
float temperature = 36.5; // 소수점 숫자 36.5
char grade = 'A';         // 글자 하나 A

    💡 팁! C 언어에서는 변수를 만들 때 반드시 자료형을 먼저 적어야 해요. "이 상자에는 정수를 넣을 거야!"라고 미리 알려주는 거죠.

    개념 3: 변수의 선언과 값 저장 ✍️¶

    비유로 시작해요.

    상자를 사용하려면 두 단계가 필요하죠? ① 상자를 준비하고 ② 물건을 넣는 것. 변수도 마찬가지예요.

    정확한 정의:

    • 선언(declaration): 변수를 만드는 것. "이런 종류의 상자를 하나 준비해줘!"
    • 대입(assignment): 변수에 값을 넣는 것. "이 상자에 이 값을 넣어줘!"

    예시로 확인:

    // 방법 1: 선언과 대입을 따로
int age;        // 상자 준비 (선언)
age = 25;       // 값 넣기 (대입)

// 방법 2: 선언과 동시에 대입 (초기화)
int age = 25;   // 상자 준비하면서 바로 값 넣기

    값은 바꿀 수 있어요! (그래서 "변수"잖아요 😄)

    int score = 80;    // 처음에 80점
score = 95;        // 95점으로 변경!

    여기서 =는 수학의 "같다"가 아니라, "오른쪽 값을 왼쪽에 넣어라"라는 뜻이에요. 프로그래밍에서는 이것을 대입 연산자라고 부릅니다.

    flowchart LR A["score = 80"] -->|"처음"| B["📦 score\n안에 80"] B -->|"score = 95 실행"| C["📦 score\n안에 95"] style B fill:#ffe0b2 style C fill:#c8e6c9

    💡 잠깐! 여기까지 이해가 잘 되셨나요? 변수는 상자, 자료형은 상자의 종류, 대입은 상자에 값 넣기. 이 세 가지만 기억하면 돼요. 어렵지 않죠? 😊

    개념 4: scanf — 사용자에게 값 입력받기 ⌨️¶

    비유로 시작합니다.

    지금까지의 printf가 프로그램이 말하는 것이었다면, scanf는 프로그램이 듣는 것이에요. 사용자가 키보드로 입력한 값을 변수에 저장해 줍니다.

    정확한 정의:

    🔑 scanf는 사용자의 키보드 입력을 받아 변수에 저장하는 함수입니다.

    기본 사용법:

    int age;
scanf("%d", &age);  // 사용자가 입력한 정수를 age에 저장

    여기서 중요한 것 두 가지:

    • %d → "정수를 입력받을 거야"라는 뜻 (서식 지정자)
    • &age → "age 상자의 주소를 알려줄게, 여기에 넣어줘!" (&는 "주소"를 의미)

    각 자료형에 맞는 서식 지정자가 있어요:

    • %d → int (정수)
    • %f → float (소수점 숫자)
    • %c → char (글자 하나)
    flowchart LR A["⌨️ 사용자가\n25를 입력"] -->|"scanf('%d', &age)"| B["📦 age\n안에 25 저장"] style A fill:#e1bee7 style B fill:#c8e6c9

    ⚠️ 주의! scanf에서 변수 이름 앞에 &를 빼먹으면 프로그램이 오류를 일으켜요. 이것은 입문자가 가장 많이 하는 실수 중 하나입니다. 꼭 기억하세요!

    🔨 따라하기: "자기소개 프로그램" 만들기¶

    배운 개념을 모두 활용해서, 사용자가 이름의 첫 글자, 나이, 키를 입력하면 자기소개를 출력하는 프로그램을 만들어 봅시다!

    Step 1: 기본 틀 만들기¶

    에디터를 열고 새 파일 introduce.c를 만드세요. 먼저 기본 구조만 작성합니다.

    #include <stdio.h>

int main(void) {

    return 0;
}

    ✅ 체크포인트: 이 구조가 자연스럽게 느껴진다면, 이전 장을 잘 배운 거예요!

    Step 2: 변수 선언하기¶

    main 함수 안에 세 가지 변수를 선언합니다. 어떤 자료형을 써야 할지 생각해 보세요.

    #include <stdio.h>

int main(void) {
    char initial;       // 이름의 첫 글자 (글자 하나 → char)
    int age;            // 나이 (정수 → int)
    float height;       // 키 (소수점 → float)

    return 0;
}

    왜 이런 자료형을 골랐을까요? - 이름의 첫 글자 → 글자 하나니까 char - 나이 → 25살, 30살처럼 정수니까 int - 키 → 175.5cm처럼 소수점이 있을 수 있으니까 float

    Step 3: 사용자에게 입력받기¶

    printf로 안내 메시지를 보여준 뒤, scanf로 값을 입력받습니다.

    #include <stdio.h>

int main(void) {
    char initial;
    int age;
    float height;

    printf("이름의 첫 글자를 입력하세요: ");
    scanf("%c", &initial);

    printf("나이를 입력하세요: ");
    scanf("%d", &age);

    printf("키(cm)를 입력하세요: ");
    scanf("%f", &height);

    return 0;
}

    💡 포인트! printf로 먼저 안내하고, scanf로 입력받는 패턴이 보이시나요? 이 "안내 → 입력" 패턴은 앞으로 계속 사용하게 될 거예요.

    Step 4: 입력받은 값 출력하기¶

    이제 저장된 값을 printf로 출력합니다.

    #include <stdio.h>

int main(void) {
    char initial;
    int age;
    float height;

    printf("이름의 첫 글자를 입력하세요: ");
    scanf("%c", &initial);

    printf("나이를 입력하세요: ");
    scanf("%d", &age);

    printf("키(cm)를 입력하세요: ");
    scanf("%f", &height);

    printf("\n=== 자기소개 ===\n");
    printf("이름 첫 글자: %c\n", initial);
    printf("나이: %d살\n", age);
    printf("키: %.1fcm\n", height);

    return 0;
}

    💡 %.1f가 뭔가요? 소수점 아래 1자리까지만 보여달라는 뜻이에요. %.2f라고 쓰면 소수점 아래 2자리까지 보여줍니다.

    Step 5: 컴파일하고 실행하기¶

    터미널에서 다음 명령어를 입력하세요:

    gcc introduce.c -o introduce
./introduce

    실행하면 이런 식으로 동작합니다:

    이름의 첫 글자를 입력하세요: J
나이를 입력하세요: 25
키(cm)를 입력하세요: 175.5

=== 자기소개 ===
이름 첫 글자: J
나이: 25살
키: 175.5cm

    ✅ 체크포인트: 여러분이 입력한 값이 그대로 출력되면 성공이에요! 🎉 축하합니다!

    Step 6: 계산 추가하기 — 10년 후 나이¶

    변수의 진짜 힘은 값을 가지고 계산할 수 있다는 것입니다. 출력 부분 아래에 한 줄을 추가해 볼게요:

        printf("키: %.1fcm\n", height);

    // 여기부터 추가!
    int future_age = age + 10;
    printf("10년 후 나이: %d살\n", future_age);

    다시 컴파일하고 실행하면:

    10년 후 나이: 35살

    age에 저장된 25에 10을 더해서 future_age라는 새 변수에 저장한 것이에요. 이렇게 변수끼리 계산도 할 수 있습니다!

    💡 여기까지 정말 잘 따라오셨어요! 변수를 선언하고, 값을 입력받고, 계산까지 했어요. 이것이 프로그래밍의 핵심 흐름입니다. 👏

    📝 전체 코드¶

    #include <stdio.h>

int main(void) {
    char initial;
    int age;
    float height;

    printf("이름의 첫 글자를 입력하세요: ");
    scanf("%c", &initial);

    printf("나이를 입력하세요: ");
    scanf("%d", &age);

    printf("키(cm)를 입력하세요: ");
    scanf("%f", &height);

    printf("\n=== 자기소개 ===\n");
    printf("이름 첫 글자: %c\n", initial);
    printf("나이: %d살\n", age);
    printf("키: %.1fcm\n", height);

    int future_age = age + 10;
    printf("10년 후 나이: %d살\n", future_age);

    return 0;
}

    ⚠️ 자주 하는 실수¶

    실수 1: scanf에서 &를 빼먹는다¶

    // ❌ 잘못된 코드
scanf("%d", age);

// ✅ 올바른 코드
scanf("%d", &age);
    • 왜 이런 일이 생기나요? printf에서는 &가 없으니까, scanf에서도 빼먹기 쉬워요.
    • 이렇게 해결해요: scanf에는 항상 &가 붙는다고 기억하세요. &는 "이 상자의 주소를 알려줄게, 여기에 값을 넣어줘"라는 의미예요.

    실수 2: 자료형과 서식 지정자가 안 맞는다¶

    // ❌ int 변수인데 %f를 사용
int age = 25;
printf("%f", age);    // 이상한 값이 출력됨!

// ✅ int 변수에는 %d를 사용
printf("%d", age);    // 25가 정상 출력
    • 왜 이런 일이 생기나요? 자료형마다 메모리에 저장되는 방식이 달라서, 잘못된 서식 지정자를 쓰면 엉뚱한 값이 나와요.
    • 이렇게 해결해요: int↔%d, float↔%f, char↔%c — 이 짝꿍을 외워두세요!

    실수 3: char 값에 큰따옴표를 쓴다¶

    // ❌ 큰따옴표 사용
char grade = "A";     // 오류 발생!

// ✅ 작은따옴표 사용
char grade = 'A';     // 정상 작동
    • 왜 이런 일이 생기나요? C 언어에서 큰따옴표("A")는 문자열(여러 글자)을 의미하고, 작은따옴표('A')는 문자 하나를 의미해요. char는 글자 하나만 담으니까 작은따옴표를 써야 합니다.
    • 이렇게 해결해요: char는 작은따옴표! 이것만 기억하세요.

    실수 4: 변수를 선언하지 않고 사용한다¶

    // ❌ 선언 없이 사용
age = 25;             // age라는 상자가 없는데 값을 넣으려 함!

// ✅ 먼저 선언 후 사용
int age;
age = 25;
    • 왜 이런 일이 생기나요? 상자를 준비하지 않았는데 물건을 넣으려는 것과 같아요.
    • 이렇게 해결해요: 변수는 반드시 선언 먼저, 사용은 그 다음!

    ✅ 스스로 점검하기¶

    다음 질문에 스스로 답해 보세요:

    • 변수가 무엇인지 한 문장으로 설명할 수 있나요?
    • int, float, char의 차이를 알고, 상황에 맞는 자료형을 고를 수 있나요?
    • int number = 10; 이 코드에서 자료형, 변수 이름, 값이 각각 무엇인지 구분할 수 있나요?
    • scanf를 사용해서 사용자 입력을 받는 코드를 직접 작성할 수 있나요?
    • scanf에서 &를 왜 붙여야 하는지 설명할 수 있나요?

    3개 이상 체크할 수 있다면, 이 장을 충분히 이해한 거예요! 전부 체크했다면 완벽합니다! 🌟

    도전 과제: 온도 변환기 🌡️¶

    사용자에게 섭씨 온도를 입력받아 화씨로 변환하는 프로그램을 만들어 보세요.

    변환 공식: 화씨 = 섭씨 × 1.8 + 32

    힌트: - 섭씨와 화씨 모두 소수점이 있을 수 있으니 float를 사용하세요 - scanf로 입력받고, 계산한 뒤, printf로 출력하면 돼요

    실행 예시: 

    섭씨 온도를 입력하세요: 36.5
화씨 온도: 97.7

    혼자 힘으로 작성해 본 뒤, 아래 정답을 확인하세요!

    정답 보기 (직접 시도한 후 클릭!) 
    #include <stdio.h>

int main(void) {
    float celsius;
    float fahrenheit;

    printf("섭씨 온도를 입력하세요: ");
    scanf("%f", &celsius);

    fahrenheit = celsius * 1.8 + 32;

    printf("화씨 온도: %.1f\n", fahrenheit);

    return 0;
}

    📌 이 장의 핵심 정리¶

    • ✅ 변수는 데이터를 저장하는 이름 붙은 메모리 공간이에요
    • ✅ 자료형은 변수에 담을 데이터의 종류를 지정해요 — int(정수), float(소수점), char(글자 하나)
    • ✅ =는 "같다"가 아니라 "오른쪽 값을 왼쪽에 넣어라"는 대입 연산자예요
    • ✅ scanf는 사용자 입력을 변수에 저장하며, 변수 앞에 반드시 &를 붙여야 해요
    • ✅ 서식 지정자 짝꿍: int↔%d, float↔%f, char↔%c
