    나만의 함수 만들기 — 함수 정의와 호출¶

    Part 3 | [1권] 코드를 깔끔하게 정리하는 힘 — 함수

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • 반환형, 함수 이름, 매개변수, 함수 본체로 구성된 함수 정의 문법을 이해할 수 있다
    • 직접 함수를 정의하고 main에서 호출하여 실행할 수 있다
    • void 함수와 값을 반환하는 함수의 차이를 구분할 수 있다
    • 함수를 활용해 코드를 깔끔하게 정리할 수 있다

    ⏱️ 예상 학습 시간: 약 2시간

    📚 미리 알아야 할 것¶

    지금까지 우리는 main 함수 안에서 변수, 조건문, 반복문을 사용하는 방법을 배웠어요. 모든 코드를 main 안에 작성했죠. 이번 장부터는 main 바깥에 나만의 함수를 만드는 법을 배웁니다. printf처럼 이름을 불러서 사용하는 함수를, 이제 우리가 직접 만들 수 있게 되는 거예요!

    💡 왜 이걸 배우나요?¶

    지금까지 작성한 프로그램을 떠올려 보세요. main 함수 안에 모든 코드를 넣었죠? 프로그램이 짧을 때는 괜찮지만, 길어지면 어떨까요?

    예를 들어, "인사말 출력"이 프로그램의 여러 곳에서 필요하다고 해봅시다. 같은 코드를 세 번, 네 번 복사해서 붙여넣어야 할까요? 😰

    함수를 만들면 한 번 작성한 코드를 이름만 불러서 반복 사용할 수 있습니다. 마치 요리 레시피 카드를 한 장 만들어두면, 그 요리가 필요할 때마다 카드를 꺼내 따라 하면 되는 것처럼요! 🍳

    📚 핵심 개념¶

    개념 1: 함수는 "나만의 레시피 카드" 🃏¶

    비유로 시작할게요.

    여러분이 카페를 운영한다고 상상해 보세요. 아메리카노를 만들 때마다 원두 갈기, 물 끓이기, 추출하기… 이 모든 과정을 매번 처음부터 생각해야 한다면 너무 비효율적이겠죠?

    그래서 레시피 카드를 만듭니다:

    • 레시피 이름: 아메리카노
    • 필요한 재료: 원두 20g, 물 200ml
    • 만드는 과정: 원두를 갈고 → 물을 끓이고 → 추출한다
    • 완성된 결과: 아메리카노 한 잔

    C 언어의 함수도 정확히 이 구조입니다!

    정확한 정의: 함수(function)란, 특정 작업을 수행하는 코드 묶음에 이름을 붙인 것입니다. 한 번 정의해 두면 이름을 불러서(호출해서) 언제든 재사용할 수 있습니다.

    graph LR A["레시피 이름<br/>→ 함수 이름"] --- B["필요한 재료<br/>→ 매개변수"] B --- C["만드는 과정<br/>→ 함수 본체"] C --- D["완성된 결과<br/>→ 반환값"]

    개념 2: 함수 정의 문법 — 네 가지 구성 요소¶

    비유로 시작할게요.

    레시피 카드를 작성할 때 빠지면 안 되는 네 가지가 있었죠? C 함수도 마찬가지로 네 가지 요소로 구성됩니다.

    정확한 정의:

    반환형 함수이름(매개변수) {
    함수 본체 (실행할 코드)
    return 반환값;
}

    각 요소를 하나씩 살펴볼게요:

    • 반환형(return type): 함수가 작업을 끝내고 돌려주는 결과의 자료형이에요. 정수를 돌려주면 int, 아무것도 돌려주지 않으면 void를 씁니다.
    • 함수 이름(function name): 변수 이름 짓듯이, 함수가 하는 일을 알 수 있게 지어요. 예: add, greet, printLine
    • 매개변수(parameter): 함수가 작업하려면 외부에서 받아야 하는 "재료"입니다. 괄호 () 안에 넣어요. 재료가 필요 없으면 비워둡니다.
    • 함수 본체(body): 중괄호 {} 안에 실제로 실행할 코드를 작성합니다.

    예시로 확인:

    int add(int a, int b) {
    int result = a + b;
    return result;
}
    • int → 반환형 (정수를 돌려줌)
    • add → 함수 이름
    • int a, int b → 매개변수 (재료 2개)
    • { ... } → 함수 본체
    • return result; → 결과를 돌려줌

    💡 잠깐! 여기까지 이해가 잘 되셨나요? "반환형이 뭐지?" 싶으시다면 걱정 마세요. 바로 다음 개념에서 자세히 설명할게요!

    개념 3: void 함수 — "그냥 해줘, 돌려줄 건 없어"¶

    비유로 시작할게요.

    카페에서 "테이블 닦기" 레시피를 생각해 보세요. 테이블을 닦으면 끝이에요. 돌려줄 결과물(완성된 음료 같은 것)이 없죠. 그냥 "닦는 행위" 자체가 목적입니다.

    정확한 정의: void 함수는 작업을 수행하기만 하고, 호출한 곳에 아무 값도 돌려주지 않는 함수입니다. void는 영어로 "비어있는"이라는 뜻이에요.

    예시로 확인:

    void greet(void) {
    printf("안녕하세요! 반갑습니다!\n");
}
    • 반환형이 void → 돌려줄 것 없음
    • 매개변수도 void → 받을 재료도 없음
    • return문이 없어요! (있어도 return;만 쓸 수 있어요)

    이 함수는 호출하면 그냥 화면에 인사말을 출력하고 끝납니다. 심플하죠? 😊

    개념 4: 값을 반환하는 함수 — "결과를 돌려줄게!"¶

    비유로 시작할게요.

    이번에는 "아메리카노 만들기" 레시피입니다. 원두와 물을 받아서 아메리카노를 만들어서 돌려줍니다. 주문한 손님에게 커피를 건네야 하니까요!

    정확한 정의: 값을 반환하는 함수는 return문을 통해 호출한 곳에 결과값을 돌려줍니다. 돌려주는 값의 자료형이 반환형과 일치해야 해요.

    예시로 확인:

    int add(int a, int b) {
    return a + b;
}

    이 함수를 호출하면:

    int sum = add(3, 5);  // sum에 8이 저장됨

    add(3, 5)가 실행되면, a에 3이, b에 5가 들어가고, a + b의 결과인 8이 return을 통해 돌아옵니다. 마치 자판기에 동전(매개변수)을 넣으면 음료(반환값)가 나오는 것과 같아요! 🥤

    graph LR A["호출: add(3, 5)"] -->|"a=3, b=5 전달"| B["함수 실행<br/>a + b 계산"] B -->|"return 8"| C["결과: sum = 8"]

    💡 잠깐! void 함수와 반환 함수의 차이가 헷갈린다면 이렇게 기억하세요: - void 함수 = 일만 하고 끝 (청소 도우미 🧹) - 반환 함수 = 일하고 결과를 가져다 줌 (배달 기사 📦)

    🔨 따라하기¶

    이제 직접 함수를 만들어 봅시다! 천천히 따라오세요. 😊

    Step 1: void 함수 만들기 — 인사말 출력¶

    가장 간단한 함수부터 시작합니다. 화면에 인사말을 출력하는 함수예요.

    #include <stdio.h>

// 함수 정의: main보다 위에 작성
void greet(void) {
    printf("=============================\n");
    printf("  안녕하세요! C 프로그래밍!\n");
    printf("=============================\n");
}

int main(void) {
    greet();  // 함수 호출!
    return 0;
}

    이 코드를 입력하고 실행해 보세요.

    ✅ 체크포인트: 화면에 구분선과 함께 인사말이 출력되면 성공이에요!

    핵심 포인트를 짚어볼게요:

    • void greet(void) → 받을 것도 없고, 돌려줄 것도 없는 함수
    • main 안에서 greet();로 호출 → 이 순간 greet 함수의 코드가 실행됨
    • 함수 정의는 main보다 위에 작성했어요 (이유는 곧 설명할게요!)

    Step 2: 같은 함수를 여러 번 호출하기¶

    함수의 진짜 힘은 재사용에 있습니다! main을 이렇게 바꿔볼까요?

    int main(void) {
    greet();
    printf("첫 번째 수업을 시작합니다.\n\n");
    greet();
    printf("두 번째 수업을 시작합니다.\n\n");
    greet();
    return 0;
}

    ✅ 체크포인트: 인사말이 세 번 출력되고, 사이사이에 수업 안내 문구가 보이면 성공!

    한 번 만들어둔 레시피 카드(greet)를 세 번 꺼내 쓴 거예요. 만약 함수 없이 했다면 printf 세 줄을 세 번 반복, 총 아홉 줄을 써야 했을 거예요. 함수 덕분에 코드가 훨씬 깔끔해졌죠? ✨

    Step 3: 매개변수가 있는 void 함수¶

    이번에는 "재료를 받는" 함수를 만들어 봅시다. 이름을 받아서 맞춤형 인사를 하는 함수예요.

    #include <stdio.h>

void greetByName(char name[]) {
    printf("%s님, 환영합니다!\n", name);
}

int main(void) {
    greetByName("철수");
    greetByName("영희");
    greetByName("민수");
    return 0;
}

    실행 결과: 

    철수님, 환영합니다!
영희님, 환영합니다!
민수님, 환영합니다!

    매개변수 name에 매번 다른 이름이 전달됩니다. 같은 함수인데 재료(이름)가 달라지니 결과도 달라져요!

    Step 4: 값을 반환하는 함수 만들기¶

    이제 결과를 돌려주는 함수를 만들어 봅시다. 두 수를 더해서 결과를 돌려주는 add 함수예요.

    #include <stdio.h>

int add(int a, int b) {
    int result = a + b;
    return result;
}

int main(void) {
    int sum;

    sum = add(10, 20);
    printf("10 + 20 = %d\n", sum);

    sum = add(7, 3);
    printf("7 + 3 = %d\n", sum);

    // 이렇게 바로 사용할 수도 있어요!
    printf("100 + 200 = %d\n", add(100, 200));

    return 0;
}

    ✅ 체크포인트: 세 개의 덧셈 결과가 올바르게 출력되면 성공!

    add(10, 20)을 호출하면 → a에 10, b에 20이 들어가고 → return 30으로 30이 돌아와서 → sum에 저장됩니다.

    Step 5: void 함수와 반환 함수를 함께 사용하기¶

    실전에서는 두 종류의 함수를 섞어서 사용합니다. 간단한 계산기를 만들어 볼게요!

    #include <stdio.h>

// 구분선을 출력하는 void 함수
void printLine(void) {
    printf("-------------------\n");
}

// 두 수를 더해서 결과를 돌려주는 함수
int add(int a, int b) {
    return a + b;
}

// 두 수를 빼서 결과를 돌려주는 함수
int subtract(int a, int b) {
    return a - b;
}

int main(void) {
    int x = 15;
    int y = 7;

    printLine();
    printf("  간단 계산기\n");
    printLine();

    printf("%d + %d = %d\n", x, y, add(x, y));
    printf("%d - %d = %d\n", x, y, subtract(x, y));

    printLine();

    return 0;
}

    ✅ 체크포인트: 구분선 사이에 "간단 계산기" 제목과 덧셈·뺄셈 결과가 출력되면 성공!

    printLine은 void 함수(일만 하고 끝), add와 subtract는 반환 함수(결과를 돌려줌). 이렇게 역할에 따라 적절한 함수를 만들면 main이 매우 깔끔해집니다. 👏

    Step 6: 함수의 위치와 선언 이해하기¶

    지금까지 함수를 main 위에 작성했어요. 그런데 main 아래에 함수를 쓰면 어떻게 될까요?

    #include <stdio.h>

int main(void) {
    greet();  // ❌ 컴파일 에러! greet를 아직 모름
    return 0;
}

void greet(void) {
    printf("안녕하세요!\n");
}

    C 컴파일러는 코드를 위에서 아래로 읽습니다. main에서 greet()를 호출하는 시점에 아직 greet가 뭔지 모르니 에러가 나는 거예요.

    해결 방법은 두 가지입니다:

    방법 1: 함수를 main 위에 작성 (지금까지 한 방법)

    방법 2: 함수 선언(프로토타입)을 위에 적어주기

    #include <stdio.h>

void greet(void);  // 함수 선언 (미리 알려주기)

int main(void) {
    greet();  // ✅ 이제 OK!
    return 0;
}

void greet(void) {  // 함수 정의 (실제 내용)
    printf("안녕하세요!\n");
}

    함수 선언은 "이런 함수가 아래에 있으니까 미리 알아둬!"라고 컴파일러에게 알려주는 역할이에요. 입문 단계에서는 방법 1(main 위에 작성)이 더 간단하니, 이 방법을 추천합니다! 😊

    💡 잠깐! 여기까지 정말 잘 따라오셨어요! 함수를 직접 만들고 호출하는 것, 생각보다 어렵지 않죠? 👍

    📝 전체 코드¶

    이 장에서 배운 내용을 종합한 프로그램입니다. 복사해서 실행해 보세요!

    #include <stdio.h>

/* --- 함수 정의 --- */

// void 함수: 구분선 출력
void printLine(void) {
    printf("========================\n");
}

// void 함수: 이름을 받아 인사
void greetByName(char name[]) {
    printf(">> %s님, 환영합니다!\n", name);
}

// 반환 함수: 두 수의 합
int add(int a, int b) {
    return a + b;
}

// 반환 함수: 두 수의 곱
int multiply(int a, int b) {
    return a * b;
}

/* --- 메인 함수 --- */

int main(void) {
    // void 함수 호출
    printLine();
    greetByName("독학러");
    printLine();

    // 반환 함수 호출
    int x = 8;
    int y = 5;

    int sum = add(x, y);
    int product = multiply(x, y);

    printf("%d + %d = %d\n", x, y, sum);
    printf("%d x %d = %d\n", x, y, product);

    printLine();
    printf("학습 완료!\n");

    return 0;
}

========================

========================

    ⚠️ 자주 하는 실수¶

    실수 1: 반환형과 return 값의 불일치¶

    int add(int a, int b) {
    printf("%d\n", a + b);
    // return을 깜빡했어요! ❌
}
    • 왜 이런 일이 생기나요? int를 반환한다고 선언했는데, return문이 없으면 컴파일러가 경고를 보냅니다. 결과값이 엉뚱해질 수 있어요.
    • 이렇게 해결해요: int 반환형 함수에는 반드시 return 값;을 써주세요!
    int add(int a, int b) {
    return a + b;  // ✅
}

    실수 2: 함수 호출할 때 괄호를 빼먹음¶

    greet;  // ❌ 호출이 아님! 아무 일도 안 일어남
    • 왜 이런 일이 생기나요? 괄호 ()가 있어야 "이 함수를 실행해!"라는 뜻이 됩니다. 괄호 없이 함수 이름만 쓰면 C에서는 함수의 주소값을 의미하는데, 에러도 안 나서 찾기 어려워요.
    • 이렇게 해결해요: 함수 호출에는 반드시 괄호를 붙이세요!
    greet();  // ✅ 올바른 호출

    실수 3: 함수 정의를 main 안에 넣음¶

    int main(void) {
    void greet(void) {  // ❌ 함수 안에 함수를 정의할 수 없어요!
        printf("안녕!\n");
    }
    return 0;
}
    • 왜 이런 일이 생기나요? C 언어에서는 함수 안에 다른 함수를 정의하는 것을 허용하지 않습니다.
    • 이렇게 해결해요: 함수 정의는 반드시 main 바깥에 작성하세요!
    void greet(void) {     // ✅ main 바깥에 정의
    printf("안녕!\n");
}

int main(void) {
    greet();
    return 0;
}
