    함수에 값을 주고받기 — 매개변수와 반환값 깊이 이해하기¶

    Part 3 | [1권] 코드를 깔끔하게 정리하는 힘 — 함수

    🎯 이 장의 학습 목표¶

    이 장을 마치면 이런 것들을 할 수 있어요!

    • 매개변수를 통해 함수에 데이터를 전달하는 원리를 그림으로 설명할 수 있다
    • return을 사용하여 함수 결과를 호출한 곳으로 돌려보낼 수 있다
    • 여러 개의 매개변수를 받는 함수를 작성할 수 있다
    • 매개변수와 반환값을 활용한 실용적인 함수를 직접 만들 수 있다

    ⏱️ 예상 학습 시간: 약 2시간

    📚 미리 알아야 할 것¶

    지난 장(10장)에서 우리는 함수의 기본을 배웠어요. 혹시 기억이 가물가물하다면 아래만 빠르게 복습해 주세요!

    • def 키워드로 함수를 만든다
    • 함수를 정의하는 것과 호출하는 것은 다르다
    • 함수를 호출하면 그 함수 안의 코드가 실행된다
    # 지난 장에서 배운 기본 함수
def say_hello():
    print("안녕하세요!")

say_hello()  # 호출 → "안녕하세요!" 출력

    이 함수는 항상 똑같은 인사만 해요. "철수에게 인사", "영희에게 인사"처럼 상황에 따라 다르게 동작하게 하려면 어떻게 해야 할까요? 바로 오늘 배울 매개변수와 반환값이 필요합니다! 🎁

    💡 왜 이걸 배우나요?¶

    지금까지 만든 함수는 마치 버튼 하나만 있는 자판기와 같았어요. 누르면 항상 같은 음료가 나오죠. 하지만 현실의 자판기는 돈을 넣고, 원하는 음료를 선택하면, 해당 음료가 나옵니다.

    프로그래밍도 마찬가지예요!

    • 매개변수 = 자판기에 넣는 돈과 선택 버튼 (함수에 주는 값)
    • 반환값 = 자판기에서 나오는 음료 (함수가 돌려주는 결과)

    이 두 가지를 익히면, 진짜로 유용한 함수를 만들 수 있게 됩니다. 계산기, BMI 측정기, 성적 처리기 등 여러분이 상상하는 거의 모든 기능을 함수로 구현할 수 있어요!

    📚 핵심 개념¶

    개념 1: 매개변수 — 함수에게 전달하는 택배 상자 📦¶

    🎪 비유로 시작해요

    택배를 보내본 적 있나요? 택배 상자에 물건을 넣어서 수신인(함수)에게 보내면, 수신인은 상자를 열어 물건을 꺼내 사용합니다. 매개변수는 바로 이 택배 상자 역할이에요.

    📖 정확한 정의

    매개변수(Parameter) 는 함수가 외부에서 값을 전달받기 위해 마련한 변수입니다. 함수를 정의할 때 괄호 () 안에 적어둡니다.

    def say_hello(name):    # name이 매개변수
    print(f"{name}님, 안녕하세요!")

    인수(Argument) 는 함수를 호출할 때 실제로 전달하는 값입니다.

    say_hello("철수")    # "철수"가 인수
say_hello("영희")    # "영희"가 인수

    ✅ 예시로 확인

    def say_hello(name):
    print(f"{name}님, 안녕하세요!")

say_hello("철수")   # 출력: 철수님, 안녕하세요!
say_hello("영희")   # 출력: 영희님, 안녕하세요!
say_hello("민수")   # 출력: 민수님, 안녕하세요!

    같은 함수인데 전달하는 값에 따라 결과가 달라져요! 이것이 매개변수의 힘입니다. 💪

    💡 헷갈리지 마세요! - 매개변수(Parameter): 함수를 만들 때 괄호 안에 쓰는 이름 → name - 인수(Argument): 함수를 부를 때 괄호 안에 넣는 실제 값 → "철수"

    일상에서는 두 용어를 혼용하기도 해요. 지금은 "함수에 값을 넘긴다"는 개념만 확실히 잡으면 됩니다!

    개념 2: 반환값 — 함수가 돌려보내는 결과물 🎁¶

    🎪 비유로 시작해요

    세탁소에 더러운 옷(매개변수)을 맡기면, 세탁소는 깨끗한 옷(반환값)을 돌려줍니다. 함수도 마찬가지예요. 값을 받아 처리한 뒤, 결과를 돌려보낼 수 있습니다.

    📖 정확한 정의

    반환값(Return Value) 은 함수가 return 키워드를 사용하여 호출한 곳으로 돌려보내는 값입니다.

    def double(number):
    result = number * 2
    return result       # 결과를 돌려보낸다!

    ✅ 예시로 확인

    def double(number):
    result = number * 2
    return result

answer = double(5)      # double 함수가 10을 돌려줌 → answer에 저장
print(answer)           # 출력: 10

print(double(7))        # 출력: 14 (바로 출력도 가능!)

    return이 있으면 함수의 결과를 변수에 저장하거나 다른 계산에 활용할 수 있어요. print()만 쓰는 것과는 완전히 다릅니다!

    ⚠️ print()와 return은 다릅니다! - print() → 화면에 보여주기만 할 뿐, 값을 돌려주지 않음 - return → 값을 호출한 곳으로 실제로 돌려줌 (저장, 계산 가능)

    개념 3: 값의 흐름 추적하기 — 그림으로 이해하기 🗺️¶

    매개변수와 반환값이 어떻게 이동하는지, 흐름을 시각적으로 확인해 봅시다.

    flowchart LR A["호출하는 코드\n answer = double(5)"] -->|"5를 전달 📦"| B["double 함수\nnumber = 5\nresult = 5 * 2"] B -->|"10을 반환 🎁"| C["answer = 10"]

    이 흐름을 더 자세히 풀어볼게요.

    sequenceDiagram participant 메인코드 participant double함수 메인코드->>double함수: double(5) 호출, 5를 전달 Note right of double함수: number = 5 Note right of double함수: result = 5 * 2 = 10 double함수-->>메인코드: return 10 (결과 반환) Note left of 메인코드: answer = 10

    흐름 정리 1. 메인 코드에서 double(5) 호출 → 숫자 5가 함수로 전달됨 2. 함수 안에서 number에 5가 담김 3. number * 2 계산 → 10이 됨 4. return result로 10을 호출한 곳으로 돌려보냄 5. 돌아온 10이 answer 변수에 저장됨

    💡 잠깐! 여기까지 이해가 잘 되셨나요? 어렵게 느껴져도 괜찮아요. 값이 "들어갔다가 나온다"는 흐름만 기억하세요! 택배를 보내고(매개변수) → 가공된 결과물을 받는다(반환값) 🎯

    개념 4: 여러 개의 매개변수 — 택배 상자를 여러 개 보내기 📦📦¶

    🎪 비유로 시작해요

    카페에서 음료를 주문할 때 "아메리카노, 아이스, 톨 사이즈"처럼 여러 정보를 함께 전달하죠? 함수도 매개변수를 여러 개 받을 수 있습니다!

    📖 정확한 정의

    매개변수를 여러 개 쓸 때는 쉼표(,)로 구분합니다.

    def add(a, b):          # 매개변수 2개
    return a + b

def introduce(name, age, city):   # 매개변수 3개
    print(f"저는 {city}에 사는 {age}살 {name}입니다.")

    ✅ 예시로 확인

    def add(a, b):
    return a + b

result = add(3, 7)
print(result)           # 출력: 10

# 매개변수의 순서가 중요합니다!
def subtract(a, b):
    return a - b

print(subtract(10, 3))  # 출력: 7  (10 - 3)
print(subtract(3, 10))  # 출력: -7 (3 - 10) ← 순서가 바뀌면 결과도 달라요!

    🔑 핵심 포인트: 함수를 호출할 때 넣는 인수의 순서가 매개변수의 순서와 일치해야 합니다!

    flowchart LR A["subtract(10, 3)"] -->|"첫 번째 값 10"| B["a = 10"] A -->|"두 번째 값 3"| C["b = 3"] B --> D["a - b = 10 - 3 = 7"] C --> D

    🔨 따라하기¶

    실제로 코드를 작성하며 감을 잡아봅시다! 🖥️

    Step 1: 기본 매개변수 함수 만들기¶

    목표: 이름을 받아 환영 메시지를 출력하는 함수를 만들어요.

    def welcome(name):
    print(f"🎉 {name}님, 환영합니다!")
    print(f"오늘도 즐거운 코딩 되세요!")

welcome("철수")
welcome("영희")

    실행 결과: 

    🎉 철수님, 환영합니다!
오늘도 즐거운 코딩 되세요!
🎉 영희님, 환영합니다!
오늘도 즐거운 코딩 되세요!

    ✅ 체크포인트: 이름만 바꿔서 호출했는데 다른 메시지가 나오면 성공!

    Step 2: return으로 결과 돌려받기¶

    목표: 두 수 중 큰 수를 찾아 돌려주는 함수를 만들어요.

    def find_bigger(a, b):
    if a > b:
        return a
    else:
        return b

bigger = find_bigger(10, 25)
print(f"더 큰 수는 {bigger}입니다.")   # 출력: 더 큰 수는 25입니다.

# 반환값을 다른 계산에 활용할 수도 있어요!
result = find_bigger(7, 3) + 100
print(result)   # 출력: 107 (7 + 100)

    ✅ 체크포인트: find_bigger(7, 3) + 100이 107로 나오면 성공! return 덕분에 결과를 다른 계산에 쓸 수 있는 거예요.

    Step 3: 여러 매개변수로 실용적인 함수 만들기¶

    목표: BMI를 계산하는 함수를 만들어요!

    BMI 공식: 체중(kg) ÷ (키(m) × 키(m))

    def calculate_bmi(weight, height_cm):
    height_m = height_cm / 100          # cm를 m로 변환
    bmi = weight / (height_m ** 2)      # BMI 계산
    return round(bmi, 1)                # 소수점 1자리로 반올림하여 반환

# 사용해 봅시다
my_bmi = calculate_bmi(70, 175)
print(f"BMI: {my_bmi}")                # 출력: BMI: 22.9

friend_bmi = calculate_bmi(55, 160)
print(f"친구의 BMI: {friend_bmi}")      # 출력: 친구의 BMI: 21.5

    ✅ 체크포인트: 몸무게 70kg, 키 175cm를 넣었을 때 22.9가 나오면 성공!

    Step 4: print와 return의 차이를 직접 체험하기¶

    목표: 이 차이를 확실히 느껴봅시다. 이것은 입문자가 가장 많이 헷갈리는 부분이에요!

    # print만 사용하는 함수
def add_print(a, b):
    print(a + b)

# return을 사용하는 함수
def add_return(a, b):
    return a + b

# 차이를 확인해 봅시다!
result1 = add_print(3, 5)     # 화면에 8이 출력됨
print(f"result1의 값: {result1}")   # 출력: result1의 값: None 😱

result2 = add_return(3, 5)    # 화면에 아무것도 안 나옴
print(f"result2의 값: {result2}")   # 출력: result2의 값: 8 ✅

# return이 있어야 이런 게 가능해요!
total = add_return(3, 5) + add_return(10, 20)
print(f"합계: {total}")            # 출력: 합계: 38

    add_print 함수는 화면에 보여주기만 하고, 실제로 값을 돌려주지 않아서 result1이 None이 됩니다. 반면 add_return 함수는 값을 돌려주므로 저장하거나 추가 계산에 활용할 수 있어요!

    💡 기억하세요: 함수 결과를 나중에 쓸 일이 있다면 반드시 return을 사용하세요!

    Step 5: 종합 실습 — 성적 등급 판정기¶

    목표: 점수를 받아 등급을 반환하는 함수를 만들어요.

    def get_grade(score):
    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    else:
        return "F"

# 사용해 봅시다
my_grade = get_grade(85)
print(f"내 등급: {my_grade}")       # 출력: 내 등급: B

# 여러 과목의 등급을 한 번에!
subjects = {"국어": 92, "수학": 78, "영어": 88}

for subject, score in subjects.items():
    grade = get_grade(score)
    print(f"{subject}: {score}점 → {grade}등급")

    실행 결과: 

    내 등급: B
국어: 92점 → A등급
수학: 78점 → C등급
영어: 88점 → B등급

    ✅ 체크포인트: 92점이 A, 78점이 C, 88점이 B로 나오면 성공! return으로 등급을 돌려받았기 때문에 이렇게 자유롭게 활용할 수 있는 거예요.

    📝 전체 코드¶

    지금까지 배운 개념을 하나로 합친 종합 예제입니다. 복사해서 바로 실행해 보세요!

    # ===== 매개변수와 반환값 종합 실습 =====

# 1. BMI 계산 함수
def calculate_bmi(weight, height_cm):
    height_m = height_cm / 100
    bmi = weight / (height_m ** 2)
    return round(bmi, 1)

# 2. BMI 상태 판정 함수
def get_bmi_status(bmi):
    if bmi < 18.5:
        return "저체중"
    elif bmi < 23:
        return "정상"
    elif bmi < 25:
        return "과체중"
    else:
        return "비만"

# 3. 두 함수를 조합하여 사용!
def health_check(name, weight, height_cm):
    bmi = calculate_bmi(weight, height_cm)
    status = get_bmi_status(bmi)
    return f"{name}님의 BMI: {bmi} ({status})"

# 실행
print(health_check("철수", 70, 175))
print(health_check("영희", 55, 160))
print(health_check("민수", 90, 170))

    실행 결과: 

    철수님의 BMI: 22.9 (정상)
영희님의 BMI: 21.5 (정상)
민수님의 BMI: 31.1 (비만)

    💡 핵심 포인트: health_check 함수 안에서 calculate_bmi와 get_bmi_status 함수를 조합해서 사용했어요! 함수의 반환값을 다른 함수의 인수로 전달할 수 있다는 것, 이것이 return의 진짜 힘입니다! 🚀

    실수 1: 매개변수 개수가 안 맞을 때¶

    def add(a, b):
    return a + b

add(1)        # ❌ TypeError: 2개를 기대하는데 1개만 줌
add(1, 2, 3)  # ❌ TypeError: 2개를 기대하는데 3개를 줌
add(1, 2)     # ✅ 올바른 호출
    • 왜 이런 일이 생기나요? 함수를 정의할 때 정한 매개변수 개수와, 호출할 때 넣는 인수 개수가 일치해야 합니다.
    • 이렇게 해결해요: 함수 정의를 확인하고, 매개변수 개수에 맞춰 인수를 전달하세요.

    실수 2: return을 빼먹고 print만 쓰기¶

    def add_wrong(a, b):
    print(a + b)      # 화면에 보여주기만 함

result = add_wrong(3, 5)    # 8이 출력되지만...
print(result * 2)            # ❌ TypeError! result는 None이라서 계산 불가
    • 왜 이런 일이 생기나요? print()는 화면 출력만 하고, 값을 돌려주지 않습니다. return이 없으면 함수는 자동으로 None을 반환해요.
    • 이렇게 해결해요: 결과를 활용해야 한다면 return을 사용하세요.
    def add_correct(a, b):
    return a + b       # ✅ 값을 돌려줌!

result = add_correct(3, 5)
print(result * 2)      # 출력: 16

    실수 3: return 뒤에 코드를 쓰기¶

    def example(x):
    return x * 2
    print("이 줄은 실행될까요?")   # ❌ 절대 실행되지 않음!
    • 왜 이런 일이 생기나요? return을 만나면 함수는 즉시 종료됩니다. 그 뒤의 코드는 영원히 실행되지 않아요.
    • 이렇게 해결해요: print()가 필요하다면 return 전에 쓰세요.
    def example(x):
    print("계산을 시작합니다!")     # ✅ return 전이라 실행됨
    return x * 2
