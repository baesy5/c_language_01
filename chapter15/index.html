    게임에 우연을 더하자 — 난수와 유용한 함수들¶

    Part 4: [2권] 게임을 더 풍성하게 — 배열과 실전 도구

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • rand() 함수를 사용하여 랜덤 숫자를 생성할 수 있다
    • srand(time(NULL))로 매번 다른 난수를 만드는 원리를 설명할 수 있다
    • 원하는 범위(예: 1~6)의 난수를 만드는 공식을 활용할 수 있다
    • 난수를 활용한 간단한 게임(주사위, 가위바위보)을 만들 수 있다

    ⏱️ 예상 학습 시간: 약 1시간 30분

    📚 미리 알아야 할 것¶

    이 장을 시작하기 전에, 이전 장들에서 배운 다음 내용이 필요합니다:

    • 변수와 자료형: int 변수에 값을 저장하는 방법
    • 반복문: for 문으로 코드를 여러 번 실행하는 방법
    • 조건문: if, else if, else로 분기 처리하는 방법
    • 함수 호출: printf() 같은 함수를 호출하는 기본 방법

    이 개념들이 기억나지 않더라도 걱정 마세요! 코드를 따라 치면서 자연스럽게 복습됩니다. 😊

    💡 왜 이걸 배우나요?¶

    여러분이 좋아하는 게임을 떠올려 보세요. 보드게임의 주사위, 카드게임의 카드 섞기, RPG에서 몬스터가 떨어뜨리는 아이템... 이 모든 것의 공통점이 뭘까요?

    바로 "매번 결과가 달라진다" 는 것입니다!

    만약 주사위가 항상 3만 나온다면? 카드가 항상 같은 순서로 나온다면? 아무도 그 게임을 하고 싶지 않을 거예요. "우연" 이야말로 게임을 재미있게 만드는 핵심 재료입니다.

    C언어에서 이 "우연"을 만들어주는 도구가 바로 난수(Random Number) 입니다. 이번 장에서 난수를 다루는 법을 배우면, 여러분의 프로그램이 매번 실행할 때마다 다른 결과를 보여주게 됩니다. 이것만으로도 프로그램이 훨씬 생동감 있어질 거예요! 🎲

    📚 핵심 개념¶

    개념 1: 난수(Random Number)란?¶

    🎲 비유로 시작해요

    주사위를 던져 보세요. 어떤 숫자가 나올까요? 1? 4? 6? 아무도 모릅니다. 던지기 전에는 결과를 예측할 수 없죠. 이렇게 "어떤 값이 나올지 미리 알 수 없는 숫자" 를 난수(Random Number) 라고 합니다.

    📖 정확한 정의

    🔑 난수(난수, Random Number): 특정한 규칙 없이 무작위로 생성되는 숫자. 프로그래밍에서는 컴퓨터가 수학적 계산을 통해 만들어내는 "의사 난수(Pseudo-random Number)"를 사용합니다.

    사실 컴퓨터는 진짜 "무작위"를 만들지 못합니다. 내부적으로 복잡한 수학 공식을 사용해서 무작위처럼 보이는 숫자를 만들어내는 거예요. 마치 아주 복잡한 규칙으로 주사위를 던지는 로봇 같은 거죠. 결과가 충분히 예측 불가능하기 때문에 우리는 이것을 "랜덤"이라고 부릅니다.

    ✏️ 예시로 확인

    게임에서 난수가 쓰이는 곳을 볼까요?

    • 🎲 주사위 굴리기: 1~6 사이의 난수
    • 🃏 카드 섞기: 카드 순서를 무작위로 배치
    • ⚔️ 전투 데미지: "10~20 사이의 데미지"처럼 범위 안에서 랜덤
    • 🎰 아이템 드롭: 30% 확률로 희귀 아이템 획득

    개념 2: rand() 함수 — 난수를 만드는 주사위¶

    🎲 비유로 시작해요

    rand() 함수는 엄청나게 많은 면을 가진 주사위라고 생각하면 됩니다. 이 주사위를 던지면 0부터 아주 큰 숫자(최소 32,767) 사이의 숫자가 하나 나옵니다.

    📖 정확한 정의

    🔑 rand(): <stdlib.h> 헤더에 정의된 함수로, 호출할 때마다 0 이상 RAND_MAX 이하의 정수를 하나 반환합니다. RAND_MAX는 시스템마다 다르지만, 최소 32,767 이상입니다.

    사용하려면 프로그램 맨 위에 #include <stdlib.h>를 써줘야 합니다.

    ✏️ 예시로 확인

    #include <stdio.h>
#include <stdlib.h>

int main(void) {
    printf("랜덤 숫자 1: %d\n", rand());
    printf("랜덤 숫자 2: %d\n", rand());
    printf("랜덤 숫자 3: %d\n", rand());

    return 0;
}

    이 프로그램을 실행하면 이런 결과가 나올 수 있어요:

    랜덤 숫자 1: 1804289383
랜덤 숫자 2: 846930886
랜덤 숫자 3: 1681692777

    잠깐, 뭔가 이상하지 않나요? 프로그램을 다시 실행해도 같은 숫자가 나옵니다! 😮 이건 다음 개념에서 해결할 거예요.

    💡 잠깐! rand()만 쓰면 매번 같은 숫자가 나오는 게 정상이에요. 놀라지 마세요! 바로 다음에 이 문제를 해결하는 방법을 배웁니다.

    개념 3: srand()와 time() — 매번 다른 결과 만들기¶

    🌱 비유로 시작해요

    rand()가 "주사위를 던지는 기계"라면, srand()는 그 기계에 넣는 "시작 씨앗(seed)" 입니다.

    같은 씨앗을 심으면 항상 같은 꽃이 피듯이, 같은 seed를 주면 항상 같은 난수 순서가 나옵니다. 다른 씨앗을 심으면? 다른 꽃이 피죠! 매번 다른 씨앗을 넣으려면 현재 시간을 씨앗으로 사용하면 됩니다. 시간은 매 초마다 바뀌니까요! ⏰

    📖 정확한 정의

    🔑 srand(seed): 난수 생성기의 시작점(seed)을 설정하는 함수. <stdlib.h>에 정의되어 있습니다.

    🔑 time(NULL): 현재 시간을 초 단위 숫자로 반환하는 함수. <time.h>에 정의되어 있습니다. 매 초마다 값이 바뀌므로 seed로 사용하기에 안성맞춤입니다.

    이 둘을 합치면 마법의 한 줄이 탄생합니다:

    srand(time(NULL));

    이 한 줄이 의미하는 것을 풀어볼게요:

    flowchart LR A["time(NULL)<br/>현재 시간을<br/>숫자로 가져옴"] --> B["srand( )<br/>그 숫자를<br/>씨앗으로 설정"] B --> C["rand( )<br/>씨앗 기반으로<br/>난수 생성"] style A fill:#FFE0B2,stroke:#FF9800 style B fill:#C8E6C9,stroke:#4CAF50 style C fill:#BBDEFB,stroke:#2196F3

    ✏️ 예시로 확인

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));  // 현재 시간으로 씨앗 설정!

    printf("랜덤 숫자 1: %d\n", rand());
    printf("랜덤 숫자 2: %d\n", rand());
    printf("랜덤 숫자 3: %d\n", rand());

    return 0;
}

    이제 실행할 때마다 다른 숫자가 나옵니다! 🎉

    [첫 번째 실행]
랜덤 숫자 1: 974523871
랜덤 숫자 2: 181627498
랜덤 숫자 3: 603276492

[두 번째 실행]
랜덤 숫자 1: 2047591203
랜덤 숫자 2: 748205163
랜덤 숫자 3: 1293847562

    ⚡ 핵심 규칙: srand(time(NULL))은 프로그램에서 딱 한 번만 호출합니다! 보통 main() 함수의 맨 처음에 한 번만 써주세요. 반복문 안에 넣으면 안 됩니다.

    개념 4: 원하는 범위의 난수 만들기¶

    🎯 비유로 시작해요

    rand()는 0부터 수만 이상의 거대한 숫자를 줍니다. 하지만 주사위 게임을 만들려면 1~6만 필요하고, 동전 던지기는 0~1만 필요하죠. 이때 나머지 연산자(%)가 등장합니다!

    큰 수를 6으로 나눈 나머지는 항상 0~5 사이입니다. 여기에 1을 더하면 1~6이 되죠!

    📖 정확한 정의

    🔑 범위 난수 공식: rand() % 범위 + 시작값 - rand() % 6은 0~5를 만듭니다 - rand() % 6 + 1은 1~6을 만듭니다

    flowchart TD A["rand() 호출<br/>예: 1804289383"] --> B["% 6 (나머지 연산)<br/>1804289383 % 6 = 3"] B --> C["+ 1 (시작값 조정)<br/>3 + 1 = 4"] C --> D["결과: 4<br/>(1~6 범위의 난수)"] style A fill:#E1F5FE,stroke:#0288D1 style B fill:#FFF3E0,stroke:#FF9800 style C fill:#E8F5E9,stroke:#4CAF50 style D fill:#F3E5F5,stroke:#9C27B0

    ✏️ 다양한 범위 예시

    일상적으로 자주 쓰는 범위들을 정리해 볼게요:

    • 주사위 (1~6): rand() % 6 + 1
    • 동전 던지기 (0 또는 1): rand() % 2
    • 1~100 사이: rand() % 100 + 1
    • 0~9 사이: rand() % 10
    • 10~50 사이: rand() % 41 + 10

    마지막 예시가 좀 헷갈리죠? 공식을 다시 볼게요:

    🔑 일반 공식: rand() % (최대값 - 최소값 + 1) + 최소값

    10~50이면: rand() % (50 - 10 + 1) + 10 → rand() % 41 + 10

    💡 잠깐! 여기까지 이해가 잘 되셨나요? 핵심은 딱 세 가지예요: 1. rand()로 큰 숫자를 만들고 2. %로 범위를 줄이고 3. +로 시작값을 맞춘다

    어렵게 느껴져도 괜찮아요. 바로 아래에서 직접 코딩하면서 익히면 훨씬 쉬워집니다! 😊

    🔨 따라하기 — 주사위 굴리기 시뮬레이터¶

    자, 이제 배운 개념들을 실제 코드로 만들어 봅시다! 주사위를 5번 굴리는 시뮬레이터를 단계별로 만들어 볼 거예요. 🎲

    Step 1: 기본 틀 만들기¶

    먼저 필요한 헤더 파일을 포함하고 main() 함수의 뼈대를 만듭니다.

    #include <stdio.h>    // printf, scanf용
#include <stdlib.h>   // rand, srand용
#include <time.h>     // time용

int main(void) {

    return 0;
}

    세 가지 헤더 파일을 모두 포함해야 한다는 점을 기억하세요:

    • stdio.h → 입출력 (printf, scanf)
    • stdlib.h → 난수 (rand, srand)
    • time.h → 시간 (time)

    Step 2: 난수 씨앗 설정하기¶

    main() 함수 시작 부분에 srand(time(NULL))을 추가합니다.

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));  // ← 이 줄을 추가!

    return 0;
}

    이 한 줄 덕분에 프로그램을 실행할 때마다 다른 결과가 나올 거예요.

    Step 3: 주사위 한 번 굴려보기¶

    rand() % 6 + 1로 1~6 사이의 난수를 만들어 출력합니다.

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));

    int dice = rand() % 6 + 1;  // 1~6 난수 생성
    printf("주사위 결과: %d\n", dice);

    return 0;
}

    ✅ 체크포인트: 실행할 때마다 1~6 사이의 숫자가 나오면 성공! 여러 번 실행해 보세요. 매번 다른 숫자가 나와야 합니다. (가끔 같은 숫자가 나올 수도 있는데, 그건 우연히 같은 거예요!)

    Step 4: 반복문으로 5번 굴리기¶

    for 문으로 주사위를 5번 굴리겠습니다.

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));

    printf("=== 주사위 5번 굴리기 ===\n\n");

    for (int i = 1; i <= 5; i++) {
        int dice = rand() % 6 + 1;
        printf("%d번째 굴리기: %d\n", i, dice);
    }

    return 0;
}

    Step 5: 합계와 평균 추가하기¶

    주사위 결과의 합계와 평균도 구해봅시다.

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));

    int sum = 0;
    int rolls = 5;

    printf("=== 주사위 %d번 굴리기 ===\n\n", rolls);

    for (int i = 1; i <= rolls; i++) {
        int dice = rand() % 6 + 1;
        printf("%d번째 굴리기: %d\n", i, dice);
        sum += dice;  // 합계에 더하기
    }

    printf("\n합계: %d\n", sum);
    printf("평균: %.1f\n", (double)sum / rolls);

    return 0;
}

    실행 결과 예시:

    === 주사위 5번 굴리기 ===

1번째 굴리기: 4
2번째 굴리기: 2
3번째 굴리기: 6
4번째 굴리기: 1
5번째 굴리기: 5

합계: 18
평균: 3.6

    ✅ 체크포인트: 합계와 평균이 정확하게 출력되면 성공! 평균은 소수점 한 자리까지 나와야 해요.

    💡 (double)sum / rolls가 뭔가요? sum과 rolls가 둘 다 int(정수)이면 나누기 결과도 정수가 됩니다. (double)을 붙여서 실수 나눗셈이 되도록 한 거예요. 예를 들어, 18 / 5는 3이지만, (double)18 / 5는 3.6이 됩니다.

    📝 전체 코드 — 랜덤 가위바위보 게임¶

    지금까지 배운 내용을 총동원해서, 컴퓨터와 가위바위보를 하는 게임을 만들어 봅시다! 이것이 이번 장의 종합 실습입니다. 🪨📄✂️

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));

    int player;     // 플레이어 선택
    int computer;   // 컴퓨터 선택
    int wins = 0;   // 플레이어 승리 횟수
    int losses = 0; // 플레이어 패배 횟수
    int draws = 0;  // 무승부 횟수
    int rounds = 3; // 총 라운드 수

    printf("╔════════════════════════════╗\n");
    printf("║   가위바위보 게임 (3판)    ║\n");
    printf("╚════════════════════════════╝\n\n");

    for (int i = 1; i <= rounds; i++) {
        printf("--- %d 라운드 ---\n", i);
        printf("가위(0), 바위(1), 보(2)를 선택하세요: ");
        scanf("%d", &player);

        // 입력 검증
        if (player < 0 || player > 2) {
            printf("잘못된 입력! 0, 1, 2 중에 골라주세요.\n\n");
            i--;  // 이번 라운드 다시 하기
            continue;
        }

        // 컴퓨터의 랜덤 선택
        computer = rand() % 3;  // 0, 1, 2 중 하나

        // 선택 결과 출력
        printf("당신의 선택: ");
        if (player == 0) printf("가위 ✌️\n");
        else if (player == 1) printf("바위 ✊\n");
        else printf("보 🖐️\n");

        printf("컴퓨터 선택: ");
        if (computer == 0) printf("가위 ✌️\n");
        else if (computer == 1) printf("바위 ✊\n");
        else printf("보 🖐️\n");

        // 승패 판정
        if (player == computer) {
            printf("결과: 무승부! 😐\n\n");
            draws++;
        } else if ((player == 0 && computer == 2) ||   // 가위 > 보
                   (player == 1 && computer == 0) ||    // 바위 > 가위
                   (player == 2 && computer == 1)) {    // 보 > 바위
            printf("결과: 당신이 이겼습니다! 🎉\n\n");
            wins++;
        } else {
            printf("결과: 컴퓨터가 이겼습니다! 😢\n\n");
            losses++;
        }
    }

    // 최종 결과
    printf("╔════════════════════════════╗\n");
    printf("║        최종 결과           ║\n");
    printf("╚════════════════════════════╝\n");
    printf("  승리: %d회\n", wins);
    printf("  패배: %d회\n", losses);
    printf("  무승부: %d회\n", draws);

    if (wins > losses)
        printf("\n  🏆 축하합니다! 당신의 승리!\n");
    else if (wins < losses)
        printf("\n  💻 컴퓨터의 승리! 다음에 다시 도전!\n");
    else
        printf("\n  🤝 무승부! 실력이 비슷하네요!\n");

    return 0;
}

    코드 핵심 포인트 해설¶

    이 코드에서 난수가 사용되는 핵심 부분을 짚어볼게요:

    1. 난수 초기화 (4행) 

    srand(time(NULL));
    프로그램 시작 시 딱 한 번! 이것이 없으면 컴퓨터가 매번 같은 선택을 합니다.

    2. 컴퓨터의 랜덤 선택 (26행) 

    computer = rand() % 3;  // 0, 1, 2 중 하나
    % 3으로 나머지를 구하면 0, 1, 2 세 가지 값만 나옵니다. 가위(0), 바위(1), 보(2)에 딱 맞죠!

    3. 승패 판정 로직 (39~47행)

    player == computer면 무승부, 그리고 플레이어가 이기는 세 가지 경우를 ||(또는)로 연결했습니다. 나머지는 모두 컴퓨터 승리!
