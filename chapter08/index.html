    Chapter 08. 여기까지 잘 왔어! — 중간 점검 미니 게임 만들기¶

    Part 2: [1권] 프로그램에 판단력 불어넣기

    🎯 이 장의 학습 목표¶

    이 장을 마치면 다음을 할 수 있습니다:

    • 지금까지 배운 변수, 조건문, 반복문을 하나의 프로그램으로 종합할 수 있다
    • rand() 함수를 사용하여 랜덤 숫자를 생성할 수 있다
    • 숫자 맞추기 게임(업다운 게임)을 처음부터 끝까지 직접 완성할 수 있다
    • 자신의 코드에서 오류를 찾고 수정하는 디버깅 과정을 경험할 수 있다

    예상 학습 시간: 약 2시간 30분

    • 변수와 자료형 (Chapter 3~4): int, char 등의 변수 선언과 값 저장
    • printf()와 scanf() (Chapter 4~5): 화면 출력과 키보드 입력
    • 조건문 if / else if / else (Chapter 6): 조건에 따라 다른 코드 실행
    • 반복문 while (Chapter 7): 조건이 참인 동안 코드 반복 실행

    💡 왜 이걸 배우나요?¶

    레고 블록을 한번 상상해 볼까요?

    지금까지 우리는 빨간 블록(변수), 파란 블록(조건문), 초록 블록(반복문)을 하나씩 배웠어요. 각각의 블록이 무엇인지, 어떻게 끼우는지를 연습했죠.

    하지만 레고의 진짜 재미는 뭔가요? 바로 여러 블록을 조합해서 하나의 작품을 만드는 것이에요!

    이번 장에서 만들 "업다운 숫자 맞추기 게임"은 바로 그런 작품입니다.

    • 컴퓨터가 비밀 숫자를 하나 정합니다 → 변수 + 랜덤 함수
    • 사용자가 숫자를 추측해서 입력합니다 → scanf()
    • 컴퓨터가 "더 높아요!" / "더 낮아요!" 힌트를 줍니다 → if / else if / else
    • 정답을 맞출 때까지 반복합니다 → while
    • 몇 번 만에 맞췄는지 횟수를 세서 알려줍니다 → 변수로 카운트
    flowchart LR A["🧱 변수"] --> E["🎮 업다운 게임"] B["🧱 입출력"] --> E C["🧱 조건문"] --> E D["🧱 반복문"] --> E E --> F["🏆 나의 첫 번째 완성 프로그램!"]

    이 게임을 완성하고 나면, "아, 나도 진짜 프로그램을 만들 수 있구나!" 하는 성취감을 느끼실 거예요. 그 감각이 앞으로의 학습에 큰 원동력이 됩니다!

    📚 핵심 개념¶

    이번 장에서 새로 배우는 개념은 딱 하나, 랜덤 숫자 생성이에요. 나머지는 기존에 배운 것들을 조합하는 연습입니다.

    개념 1: 랜덤 숫자 만들기 — rand()와 srand()¶

    1) 비유로 시작 🎲¶

    주사위를 던진다고 생각해 보세요. 주사위를 던지면 1~6 중 어떤 숫자가 나올지 모르죠? 컴퓨터에게도 "주사위를 던져!"라고 시킬 수 있어요. 그게 바로 rand() 함수예요.

    그런데 한 가지 문제가 있어요. 컴퓨터는 진짜 랜덤이 아니라 가짜 랜덤(의사 난수)을 만들어요. 마치 매번 같은 순서로 카드가 나오는 마술 카드 덱처럼, 프로그램을 실행할 때마다 똑같은 숫자가 나올 수 있어요.

    이걸 해결하려면 "씨앗(seed)"을 바꿔줘야 해요. 마치 카드 덱을 섞는 것처럼요. 그 역할을 하는 게 srand() 함수이고, 씨앗으로는 현재 시간을 사용합니다. 현재 시간은 매 순간 변하니까, 실행할 때마다 다른 숫자가 나오게 되는 거예요!

    2) 정확한 정의 📖¶

    • rand(): 0부터 매우 큰 수(보통 32767) 사이의 정수를 하나 돌려주는 함수
    • srand(seed): rand()가 만들어내는 숫자 순서를 결정하는 씨앗을 설정하는 함수
    • time(NULL): 현재 시간을 숫자(초 단위)로 돌려주는 함수. 씨앗으로 사용하기 딱 좋아요

    필요한 헤더 파일:

    • rand(), srand() → #include <stdlib.h>
    • time() → #include <time.h>

    3) 예시로 확인 💻¶

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));  // 현재 시간으로 씨앗 설정 (프로그램에서 딱 한 번!)

    int random_number = rand() % 100 + 1;  // 1~100 사이의 랜덤 숫자

    printf("랜덤 숫자: %d\n", random_number);

    return 0;
}

    rand() % 100 + 1이 뭔가요?

    이 부분이 핵심이에요! 차근차근 풀어볼게요.

    • rand()는 0~32767 같은 아무 숫자를 돌려줘요
    • % 100은 100으로 나눈 나머지를 구해요 → 결과는 0~99
    • + 1을 하면 → 결과는 1~100 🎯
    flowchart LR A["rand() → 예: 28473"] --> B["% 100 → 73"] B --> C["+ 1 → 74"] C --> D["결과: 1~100 사이 숫자!"]

    💡 기억 포인트! srand(time(NULL))은 프로그램 시작할 때 딱 한 번만 호출하면 돼요. 반복문 안에 넣으면 안 됩니다!

    개념 2: 시도 횟수 세기 — 카운터 변수¶

    1) 비유로 시작 🔢¶

    볼링장에 가면 점수판에 몇 프레임을 던졌는지 기록되잖아요? 프로그래밍에서도 "몇 번째인지"를 세고 싶을 때가 많아요. 이럴 때 카운터 변수를 사용합니다.

    2) 정확한 정의 📖¶

    카운터 변수란 특정 동작이 몇 번 일어났는지 세기 위해 사용하는 변수예요.

    • 처음에 0으로 시작
    • 해당 동작이 일어날 때마다 1씩 증가

    3) 예시로 확인 💻¶

    int tries = 0;  // 시도 횟수를 0으로 시작

// 사용자가 숫자를 입력할 때마다...
tries = tries + 1;  // 또는 tries++; 라고 써도 같아요!

printf("현재 %d번째 시도입니다.\n", tries);

    tries++는 tries = tries + 1을 짧게 쓴 것이에요. C 언어에서 아주 자주 사용하는 표현이니 익숙해지면 좋아요!

    개념 3: 게임 루프 — while문으로 게임의 심장 만들기¶

    1) 비유로 시작 🔄¶

    게임을 떠올려 보세요. 대부분의 게임은 이런 패턴이에요:

    1. 플레이어가 행동한다 (숫자 입력)
    2. 게임이 결과를 알려준다 (힌트 제공)
    3. 게임이 끝났는지 확인한다 (정답 맞춤?)
    4. 안 끝났으면 → 1번으로 돌아간다

    이것이 바로 게임 루프예요! 우리가 배운 while 반복문이 이 역할을 해줍니다.

    2) 예시로 확인 💻¶

    int correct = 0;  // 정답을 맞췄는지 표시 (0: 아직, 1: 맞춤)

while (correct == 0) {
    // 1. 사용자 입력 받기
    // 2. 힌트 제공하기
    // 3. 정답이면 correct = 1로 바꾸기 → 반복 탈출!
}

    개념 4: 조합의 힘 — 모든 블록을 하나로¶

    이번 개념은 새로운 문법이 아니에요. 지금까지 배운 것들을 어떻게 조합하는지를 정리하는 거예요.

    업다운 게임에 필요한 재료를 정리해 볼게요:

    • 변수: 비밀 숫자, 사용자 입력, 시도 횟수
    • 랜덤 함수: 비밀 숫자를 정하는 데 사용
    • 입출력: printf()로 힌트 출력, scanf()로 숫자 입력
    • 조건문: 입력한 숫자가 정답보다 큰지, 작은지, 같은지 판단
    • 반복문: 정답을 맞출 때까지 계속 반복

    🔨 따라하기: 업다운 게임 만들기¶

    자, 이제 진짜로 게임을 만들어 봅시다! 한꺼번에 전체 코드를 보면 어려울 수 있으니, 단계별로 하나씩 쌓아 올릴 거예요. 마치 레고 조립 설명서처럼요! 🧱

    Step 1: 프로젝트 뼈대 만들기 🦴¶

    먼저 가장 기본적인 뼈대를 만들어요. 아직 게임 로직은 없고, 필요한 헤더와 main 함수만 준비합니다.

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    printf("=========================\n");
    printf("  업다운 숫자 맞추기 게임\n");
    printf("=========================\n");
    printf("1~100 사이의 숫자를 맞춰보세요!\n\n");

    return 0;
}

    이 코드에서 확인할 것:

    • #include <stdlib.h> → rand(), srand() 사용을 위해
    • #include <time.h> → time() 사용을 위해
    • 게임 시작 화면이 보기 좋게 출력되는지 확인

    ✅ 체크포인트: 실행하면 게임 타이틀이 예쁘게 출력되어야 해요!

    Step 2: 비밀 숫자 생성하기 🎲¶

    이제 컴퓨터가 비밀 숫자를 정하도록 해볼게요.

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    // --- 게임 준비 ---

    입력받은 숫자와 비밀 숫자를 비교해서 힌트를 줄 차례예요!

    #include <stdio.h>

    Step 5: while 반복문으로 게임 루프 완성하기 🔄¶

    지금까지는 한 번만 물어보고 끝나죠? 이제 while 반복문으로 정답을 맞출 때까지 계속 반복하도록 만들어요. 이것이 게임의 핵심이에요!

    #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand(time(NULL));
    int secret = rand() % 100 + 1;
    int guess;
    int tries = 0;

    printf("=========================\n");
    printf("  업다운 숫자 맞추기 게임\n");
    printf("=========================\n");
    printf("1~100 사이의 숫자를 맞춰보세요!\n\n");

    // printf("[디버깅] 비밀 숫자: %d\n\n", secret);  // 이제 주석 처리!

    // --- 게임 루프 ---
    while (1) {   // 무한 반복! (안에서 break로 탈출)
        printf("[시도 %d] 숫자를 입력하세요: ", tries + 1);
        scanf("%d", &guess);
        tries++;

        if (guess < secret) {
            printf("⬆️ 더 높은 숫자예요! (UP!)\n\n");
        } else if (guess > secret) {
            printf("⬇️ 더 낮은 숫자예요! (DOWN!)\n\n");
        } else {
            printf("\n🎉🎉🎉 정답입니다! 🎉🎉🎉\n");
            printf("비밀 숫자 %d를 %d번 만에 맞추셨어요!\n", secret, tries);
            break;   // 반복문 탈출!
        }
    }

    printf("\n게임을 플레이해 주셔서 감사합니다!\n");

    return 0;
}

    새로 추가된 것:

    • while (1) → 무한 반복이에요! 1은 항상 참(true)이니까, 안에서 break를 만나야만 탈출할 수 있어요
    • break → 정답을 맞추면 while 반복문을 즉시 빠져나감
    • printf("[시도 %d]", tries + 1) → 몇 번째 시도인지 안내
    • 디버깅 출력을 주석 처리 → 이제 진짜 게임처럼!

    💡 while (1)이 무섭지 않아요! "무한 반복이면 영원히 돌지 않나요?"라고 걱정될 수 있어요. 걱정 마세요! 정답을 맞추면 break가 실행되어 반복문을 깔끔하게 탈출합니다. 게임에서 자주 쓰는 패턴이에요.

    ✅ 체크포인트: 실행해서 여러 번 숫자를 입력해 보세요. 힌트를 따라가면 정답에 도달하고, 축하 메시지와 시도 횟수가 출력되어야 해요!
